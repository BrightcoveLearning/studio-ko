Brightcove 플레이어는 인터페이스 요소가 다양한 언어로 표시되도록 현지화 할 수 있습니다. 플레이어 현지화 속성이 구성되지 않은 경우 인터페이스가 영어로 표시됩니다.

플레이어로컬라이제이션을구성하려면 다음 단계를 따르십시오.

  1. 플레이어모듈을 엽니다.
  2. 플레이어를 클릭하여 플레이어 속성을 엽니 다.
  3. 왼쪽 탐색 메뉴에서플레이어 정보를 클릭합니다.
  4. 현지화필드에 언어에 대한 두 글자 언어 코드를 입력하고 Enter 키를 누릅니다. 입력 할 때 언어 목록이 필터링됩니다. 값을 클릭하여 선택할 수도 있습니다.
    참고: 지원되는 언어 목록은 Video.js 언어페이지에서 찾을 수 있습니다.
  5. 적절하게 추가 언어를 추가하십시오. 잘못된 언어 코드를 입력하면 빨간색으로 표시됩니다.
  6. 저장을 클릭합니다 .

플레이어에 대한 변경 사항을 게시해야합니다.

언어 확인

현지화 된 언어를 사용하려면 브라우저에서 기본 언어를 설정하십시오. 정확한 단계는 브라우저 및 운영 체제에 따라 다릅니다. 브라우저의 기본 언어가 영어이거나 사용할 수없는 언어 인 경우 기본 또는 영어 텍스트가 표시됩니다. 브라우저에는 완벽하게 표준화 된 로케일 감지 기능이 없으므로 약간 복잡 할 수 있습니다. 일반적으로 브라우저는 다음 순서로 원하는 언어를 결정합니다.

  • 포함 페이지에<html>요소에 lang 속성이 있는지 확인합니다.
  • 브라우저 별 로케일 구성 확인
  • 영어로 돌아 가기
참고: 이 기술은 고급 (인 페이지) 코드 구현에서만 작동합니다. 표준 구현이 iframe으로 로드되므로 브라우저 언어 설정이 해당 값으로 흐르지 않습니다.

HTML 속성을 사용하는 경우 브라우저나 컴퓨터에서 기본 언어를 기본값으로 설정할 필요가 없습니다. 플레이어가lang속성의 언어를 사용하기 때문입니다.

<html lang = "es">

또한 iframe을 사용하는 경우 쿼리 매개 변수를 플레이어 URL에 추가해야합니다.
