비디오가 끝날 때 최종 화면이 표시되도록 플레이어를 구성 할 수 있습니다. 최종 화면은 재생 및 공유 아이콘을 표시하도록 구성하거나 HTML을 포함하도록 사용자 정의 할 수 있습니다. 사용자 지정 최종 화면은 아래와 같습니다.

시청자가 비디오를 재생하거나 인기있는 소셜 미디어 사이트에 비디오를 공유 할 수 있도록 아이콘이 표시됩니다.

참고: 플레이어 최종 화면이 구성된 경우 플레이어가 기본 전체 화면 모드인 경우 iOS 장치에 표시되지 않습니다.

플레이어 종료 화면을 구성하려면 다음 단계를 따르세요.

  1. 딸깍 하는 소리선수탐색 헤더에서 플레이어 모듈을 엽니다.
  2. 구성하려는 플레이어의 링크를 클릭하십시오.
  3. 왼쪽 탐색 메뉴에서최종 화면을 클릭합니다.
  4. 플레이어 종료 화면옵션을 선택합니다. 다음 옵션을 사용할 수 있습니다.
    • 없음-최종 화면이 표시되지 않습니다.
    • 사회적인 -동영상이 종료되면 소셜 공유 대화 상자 표시
    • 커스텀 -동영상 종료시 사용자 지정 최종 화면 표시
참고: 플레이어에 대한 변경 사항은 게시된 플레이어에게 나타나는 데 최대 5분이 소요될 수 있습니다.

소셜 최종 화면

소셜옵션을 선택하면 비디오가 끝날 때 소셜 공유 대화 상자가 표시됩니다.

소셜 공유 종료 화면을 사용하려면 플레이어의 소셜 공유를 활성화해야합니다. 소셜옵션을 선택하면 소셜 공유가 자동으로 활성화됩니다. 표시되는 소셜 공유 옵션은 소셜 설정의 일부로 구성된 항목을 기반으로합니다. 소셜 설정에 대한 자세한 내용은 다음을 참조하십시오. 소셜 구성 플레이어 속성 .

맞춤 최종 화면

사용자 정의옵션을 선택하면 사용자 정의 HTML을 사용하여 재생 및 공유 아이콘 위에 추가 텍스트 또는 이미지를 표시할 수 있습니다. 사용자 정의 최종 화면을 구성하려면 [ 사용자 정의 ] 를 선택하고 [HTML에 붙여넣기] 를 선택하여 [ 사용자 정의 HTML]속성에 표시합니다.

참고: 사용자 정의 HTML에 < video> html 요소를 추가하여 비디오를 사용할 수 있습니다.
참고: 공유 아이콘이 최종 화면에 나타나려면 공유를 활성화해야 합니다.
참고: 사용자 지정옵션이 선택되어 있고 HTML이 제공되지 않으면 비디오가 끝날 때 재생 버튼이 표시됩니다.
