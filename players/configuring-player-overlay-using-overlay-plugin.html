NS 위에 까는 것플러그인을 사용하면 텍스트와 이미지를 모두 오버레이로 사용할 수 있습니다. NS 위에 까는 것오버레이가 나타나는 시기와 기간에 대해서도 플러그인을 구성할 수 있습니다. 플레이어에 간단한 이미지 오버레이를 추가하려면 플레이어 스타일 옵션을 사용하십시오. 보다스타일링 선수정보를 위해.

오버레이는 플레이어 위에 나타나는 간단한 메시지 또는 이미지 일 수 있습니다. 이 플레이어에서 Brightcove 로고는 비디오의 6 초 지점까지 2 초 지점에서 플레이어의 오른쪽 하단에 표시됩니다.

다음을 기반으로 오버레이 표시를 제어 할 수 있습니다.

  • 플레이어 이벤트 -재생, 일시 중지 또는 사용자 지정 이벤트와 같은 플레이어 이벤트를 기반으로 오버레이 표시
  • 시간 간격 -동영상 재생 중 지정된 시간에 오버레이 표시

오버레이는 다음을 사용하여 구성됩니다. 위에 까는 것플러그인. 플러그인을 구성 할 때 JavaScript URL, CSS 파일, 플러그인 이름 및 플러그인 옵션을 제공해야합니다.

참고: 프로그래밍 방식으로 플레이어 오버레이를 구성하는 방법에 대한 자세한 내용은개발자 문서를 참조하십시오 .

위에 까는 것플러그인 옵션

플러그인 옵션은 오버레이가 나타나는시기, 사라지는시기, 표시 할 항목 및 표시 할 위치를 구성하는 데 사용됩니다. 다음 플러그인 옵션 (JSON 형식)이 지원됩니다.

  • start -오버레이 표시시기를 정의합니다. 값은 문자열 또는 숫자 일 수 있습니다. 숫자 인 경우 비디오 재생에서 해당 시간 (초)이 경과하면 오버레이가 표시됩니다. 값이 문자열이면 브라이트코브 플레이어 이벤트 이름 (예play : pause또는) 으로 해석됩니다. ended . 모든 플레이어 이벤트 목록은 Player API에 있습니다 .
  • end -오버레이를 숨길시기를 정의합니다. 값은 문자열 또는 숫자 일 수 있습니다. 숫자 인 경우 비디오 재생에서 해당 시간 (초)이 지나면 오버레이가 숨겨집니다. 값이 문자열이면 브라이트코브 플레이어 이벤트 이름 (예play : pause또는) 으로 해석됩니다. ended . 모든 플레이어 이벤트 목록은 Player API에 있습니다 .
  • content -오버레이로 표시 할 내용을 정의합니다. 값은 문자열 또는 DOM 개체 일 수 있습니다. 문자열, HTML 요소 (이미지) 또는 DOM DocumentFragment를 전달할 수 있습니다.
  • align -오버레이를 표시 할 위치를 정의합니다. 다음 값이 지원됩니다. top-left, top, top-right, right, bottom-right, bottom, bottom-left, left

다음 샘플 옵션은 비디오의 2 초 지점에서 시작하여 비디오가 6 초 지점을 지나면 끝나는 Brightcove 로고 이미지를 표시합니다.


{
  "overlays": [
    {
      "start": 2,
      "end": 6,
      "content": "<a href='https://www.brightcove.com' target='_blank'><img src='http://solutions.brightcove.com/bcls/assets/images/brightcove-logo.png'></a>",
      "align": "bottom-right"
    }
  ]
}

플레이어 오버레이 구성

구현하려면위에 까는 것플레이어 모듈을 사용하여 플러그인하려면 다음 단계를 따르세요.

  1. 플레이어 모듈을 열고 플레이어를 찾습니다.
  2. 플레이어의 링크를 클릭하여 플레이어 속성을 엽니 다.
  3. 딸깍 하는 소리플러그인왼쪽 탐색에서
  4. 플러그인추가 > 사용자 지정 플러그인을 클릭합니다 .
  5. 를 위해플러그인 이름 , 시작하다위에 까는 것 .
  6. 자바스크립트 URL의경우 다음을 입력합니다. 
    //players.brightcove.net/videojs-overlay/2/videojs-overlay.min.js
  7. CSS URL의 경우다음을 입력합니다. 
    //players.brightcove.net/videojs-overlay/2/videojs-overlay.css
  8. 옵션 (JSON) 을 입력합니다 . 아래 예는 플레이어 하단에 2 초 표시에서 시작하여 6 초 표시에서 끝나는 클릭 가능한 이미지를 표시합니다. 
    {
	  "overlays": [
	    {
	      "start": 2,
	      "end": 6,
	      "content": "<a href='https://www.brightcove.com' target='_blank'><img src='http://solutions.brightcove.com/bcls/assets/images/brightcove-logo.png'></a>",
	      "align": "bottom-right"
	    }
	  ]
	}
  9. 플레이어저장및게시를 클릭합니다. 게시 된 플레이어에 변경 사항이 표시되는 데 최대 5 분이 걸릴 수 있습니다.

플레이어에 대해 여러 오버레이를 구성 할 수 있습니다. 자세한 내용은개발자 설명서를 참조하십시오 .

CSS를 사용하여 플레이어 오버레이의 스타일을 지정하는 방법의 예는심벌 마크위에 까는 것플러그인개발자 문서.
