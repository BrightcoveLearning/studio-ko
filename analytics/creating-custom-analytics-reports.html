보고 시간대

사용자 정의 보고서의 데이터는 계정의보고 시간대 오프셋을 사용하여 표시됩니다. 보고 시간대는관리자  > 계정 설정 > 기본 보고 시간대로 이동하여 Video Cloud Studio 홈 페이지에서 구성할 수있습니다. 계정 관리자 만 계정 설정에 액세스 할 수 있습니다.

보고 시간대는보고 데이터의 일 경계를 계산하는 데 사용됩니다. 보고 시간대 설정을 변경하면 향후 데이터에만 영향을 미치며 변경 사항은 소급 적용되지 않습니다. 이 설정을 변경하면 변경 사항이 적용된 날짜 경계를 볼 때 데이터가 평평하거나 급증 할 수 있습니다. 이 설정에 대한 업데이트는 즉시 적용되지 않을 수 있으며 분석 데이터는 시스템이 변경 사항을 처리 할 때까지 이전 설정을 계속 참조 할 수 있습니다.

기본 보고 시간대가 적용됩니다. 오직 Analytics 모듈 및 ~ 아니다설계상 기술적으로 분리된 서비스인 Live Analytics에

계정 선택

여러 Video Cloud 계정에 액세스 할 수있는 경우 현재 Video Cloud 계정의 데이터를 포함하거나 여러 계정을 선택할 수 있습니다.

모든 계정을 포함하는 보고서를 만들고 해당 계정이 다른 시간대를 사용하는 경우 데이터는 해당 계정의 시간대를 사용하여보고됩니다.

보고서 유형 선택

보고서 유형을 선택합니다.

  • 한 번 -보고서는 즉시 한 번 실행됩니다.
  • 반복 - 보고서는 매일, 매주 또는 매월 실행되도록 예약할 수 있습니다.
    메모: 반복 보고서는 분석 조정이 수행된 후 (일반적으로 보고 기간 종료 후 3~5일 후) 실행됩니다.

일회성 보고서

일회성을 선택하면 보고서가 한 번 실행됩니다. 달력 컨트롤이 나타나고 분석 데이터에 적용 할 기간을 지정할 수 있습니다.

팁: 보고서에 찾고있는 데이터가 포함되어 있는지 확인하려면 더 작은 기간을 선택하여 시작하세요. 보고서 데이터를 확인했으면 필요에 따라 날짜 범위를 확장합니다.

반복 보고서

반복을 선택하면 보고서가 반복되는 방식으로 실행됩니다. 보고서 선택빈도 :

  • 매일 -보고서는 하루 종일 포함됩니다.
  • 주간 -보고서는 한 주 전체를 다루며 한주는 월요일부터 일요일까지로 정의됩니다.
  • 월간 간행물 -보고서는 한 달 전체를 다룹니다.
  • Enterprise 계정은 반복 보고서 20 개로 제한되고 Professional 계정은 5 개로 제한됩니다.
  • 반복 보고서는 사용자와 연결되며 사용자가 액세스 할 수있는 모든 계정에서 사용할 수 있습니다.
  • 반복 보고서는 분석 조정이 발생한 후, 일반적으로보고 기간 종료 후 3 ~ 5 일 후에 실행됩니다.

데이터 그룹 선택

데이터 그룹은 보고서에 포함될 수있는 데이터 필드를 결정합니다. 에 나타나는 데이터 필드데이터 필드섹션은 선택한 데이터 그룹을 기반으로 합니다. 여러 데이터 그룹을 선택할 수 있지만 일부 조합은 유효하지 않습니다. 데이터 그룹이 선택되면 사용 가능한 데이터 그룹 목록이 업데이트되어 선택할 수있는 유효한 그룹 만 포함됩니다.

노트:
  • Date데이터 그룹은 다른 차원과 결합할 수 없습니다.
  • 전달된바이트데이터 항목은 플레이어 보고서에서 더 이상 사용할 수 없으며데이터 그룹또는퍼포먼스 데이터에서 플레이어 항목을 선택하면 회색으로 표시됩니다.열.
  • 각 차원과 함께 반환되는 데이터 필드 목록을 보려면이 문서를 참조하십시오 .

보고서 필터 추가

반환되는 데이터의 양을 제한하기 위해 필터를 추가 할 수 있습니다. 여러 필터를 사용하는 경우 AND 연산이 사용됩니다. 다음 예에서는비디오 클라우드 기본 태그가 지정되어있고 Product=Video Cloud 사용자 지정 필드가 있는 비디오가반환됩니다.

참고: 필터는 일치하는 데이터 그룹도 선택한 경우에만 보고서에 표시됩니다. 예를 들어 비디오 태그에서 필터링하려면 비디오 차원을 선택해야 합니다.

필터를 추가하려면 단추. 필터를 제거하려면 . 다음 값을 필터로 사용할 수 있습니다.

  • 비디오 태그 -하나 이상의 비디오 태그
    • “금융 서비스" (따옴표 포함) 를 입력하면금융 서비스 태그가 지정된 동영상이 반환됩니다 .
    • 여러 태그를 사용하는 경우 모든 태그가있는 동영상이 반환됩니다. 입장 "금융 서비스", 회계태그가 지정된 모든 동영상을 반환합니다. 금융 서비스그리고회계 .

  • 사용자 정의 필드 -계정에있는 하나 이상의 사용자 정의 필드

  • 플레이어 -필터 값은 검색중인 컨텍스트에 따라 다릅니다.

데이터 필드 선택

데이터 필드섹션은 보고서에 포함될 데이터 값을 선택하는 데 사용됩니다. 사용 가능한 데이터 필드 목록은 선택한 차원을 기반으로합니다.

선택한 계정이 사용자 정의 필드로 구성된 경우 선택할 수도 있습니다. 보고서에 모든 계정이 포함 된 경우 해당 계정의 모든 사용자 정의 필드가 표시됩니다.

참고: 사용률 데이터의 경우 라이브 대역폭이 아닌 라이브 스트리밍된 실시간 초 (초) 에 대해서만 보고합니다.

보고서 미리보기

NS 보고서 미리보기섹션은 선택되어 채워진 데이터 열을 보여줍니다. 샘플 데이터 . 보고서에서 해당 데이터 항목을 제거하려면 휴지통 아이콘을 클릭하십시오.

참고: 보고서 미리 보기에 표시되는 데이터는 Video Cloud 계정의 실제 데이터가 아니라샘플 데이터입니다. 또한 미리보기의 정렬 순서는 다운로드 한 보고서에서 데이터가 정렬되는 방식에 영향을주지 않습니다. 다운로드한 보고서의 데이터는 타사 스프레드시트 응용 프로그램을 사용하여 정렬할 수 있습니다.

배달 옵션 구성

보고서에 의미있는 이름을 지정하여 시작하십시오. 이는 반복 보고서를 작성할 때 특히 중요합니다.

형식섹션은 다운로드되는 보고서 데이터의 형식을 선택하는 데 사용됩니다.

  • CSV-쉼표로 구분 된 값
  • XLSX-Microsoft Excel

보고서가 완료되면 이메일 알림을 사용하여 자신과 조직의 다른 사용자에게 알릴 수 있습니다. 보고서가 준비되면전자 메일을확인하면 보고서가 완료되면 보고서 요청자에게 전자 메일이 전송됩니다. 다른 사람에게알림텍스트 필드에 이메일 주소를 추가하여 추가 사용자에게 알림을 받을 수 있습니다. 보고서가 완료되면 보고서 링크가 포함 된 이메일을 받게됩니다.

보고서 다운로드 링크는 90 일 동안 유효합니다.

보고서 요청

보고서요청 (일회성 보고서의 경우) 또는 보고서예약 (반복 보고서의 경우) 버튼을 클릭하여 보고서를 실행합니다. 보고서가 일회성 보고서 인 경우 보고서가 완료되면 다운로드됩니다. 되풀이 보고서의 경우 보고서가 대기열에 추가되고 데이터가 준비되면 실행됩니다.
