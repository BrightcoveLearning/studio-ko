ㅏ

액티브 미디어
계정의 활성 동영상 수입니다. 이 데이터는 실시간이 아니며 하루에 한 번 비디오 라이브러리에서 동기화됩니다. 또한 "실시간"분석 데이터 (3 일 미만)를보고있는 경우이 데이터가 정확하지 않을 수 있습니다.
Analytics API
개발자가 Video Cloud 비디오에 대한 분석 지표를 JSON 형식으로 직접 검색 할 수있는 REST 기반 애플리케이션 프로그래밍 인터페이스입니다. NS 애널리틱스 API Video Cloud Studio에서 Analytics UI에 대한 데이터를 제공합니다.

비

총 대역폭
비디오, 이미지 및 플레이어로드를 제공하는 데 사용되는 총 대역폭입니다. 대역폭 데이터의 일부는 CDN과 같은 외부 소스에서 가져 오기 때문에 지연됩니다. 일반적으로 정보는 48 시간 이내에 완료되지만 경우에 따라 하루가 최대 5 일이 될 수도 있습니다.
브라우저 유형
브라우어 유형. 일반적인 값은 다음과 같습니다.
  • 크롬
  • 원정 여행
  • 파이어 폭스
  • 즉
전달 된 바이트
비디오 / 오디오 데이터 (재생 여부에 관계없이 버퍼링 된 데이터 포함), 포스터 및 썸네일 이미지, 비디오 플레이어와 관련된 모든 데이터를 포함하여 시청자에게 전달 된 총 데이터 (바이트)입니다.

트래픽 소스
시청자가 플레이어 페이지에 도달 한 방법을 나타냅니다. 가능한 값은 다음과 같습니다.
  • 추천-사용자가 동영상 링크를 따라갔습니다.
  • 직접-사용자가 직접 링크를 입력했거나 책갈피를 사용했거나 이메일의 링크를 사용했습니다.
  • 자연 검색-사용자가 검색을 수행하고 결과를 클릭하여 동영상으로 이동
  • 유료 검색-사용자가 검색을 수행하고 동영상을보기 위해 유료 결과를 클릭했습니다.
  • 보안 검색 - Google 계정으로 로그인하고 검색을 수행한 후 결과를 클릭한 사용자 (유료 또는 유기적, 이 경우 검색 정보는 포함되지 않음)
