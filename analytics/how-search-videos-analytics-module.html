빨리 찾기 사용

Analytics 모듈의 빠른 찾기 기능을 사용하여 특정 비디오를 검색 할 수 있습니다. 검색 기준을 입력하면 빠른 찾기가 비디오 이름, 간단한 설명, 자세한 설명, 태그, 비디오 ID 및 사용자가 가질 수있는 모든 사용자 지정 텍스트 필드를 검색합니다. 검색 필드 아래에 일치하는 목록이 표시됩니다.

참고삭제동영상은 검색 기능을 통해 검색할 수 없지만 해당 동영상에 대한 분석은 유지합니다. 성능 보고서를 확장/탐색하고 브라우저 검색 기능을 사용하거나 보고서를 다운로드하고 스프레드시트 앱에서 검색하여 찾을 수 있습니다.

일치하는 링크를 클릭하면 "컨텍스트에있는"보고서가 열립니다. 예를 들어 계정에 대한 참여를 확인한 다음 빠른 찾기를 사용하여 특정 비디오를 찾는 경우 비디오 링크를 클릭하면 참여 보고서가 열립니다. 실적 보고서를보고있는 경우 링크를 검색하고 클릭하면 실적 보고서가 열립니다.

기본 검색

한정자없이 검색 값을 제공하면 다음 필드에 대해 텍스트 검색이 실행됩니다.

  • 이름
  • 간단한 설명
  • 장황한 설명
  • 태그
  • 비디오 ID
  • 참조 ID
  • 모든 사용자 정의 필드

여러 단어를 일치 시키려면 용어를 따옴표로 묶으십시오.

고급 검색

분석 모듈은 다음 기능도 제공합니다. 특정 필드 검색 , 날짜를 기준으로 검색그리고특정 용어를 포함하거나 제외 .

특정 필드 검색

특정 필드에서 지정된 값을 검색하려면 다음 구문을 사용하십시오.

  • field:value to search

다음 필드가 지원됩니다.

들 지원되는 값
name 문자열 또는 인용 된 문자열
text 문자열 또는 따옴표로 묶인 문자열 ( name , short description , 의 값long description )
tags 문자열 또는 인용 된 문자열
reference_id 문자열 또는 인용 된 문자열
state ACTIVEINACTIVE , DELETED , PENDING
updated_at 보다특정 날짜 검색날짜 형식의 경우
created_at 보다특정 날짜 검색날짜 형식의 경우
published_at 보다특정 날짜 검색날짜 형식의 경우
complete true또는false

정확히 일치 시키거나 여러 단어를 일치 시키려면 용어를 따옴표로 묶으십시오.

  • "live"
  • "live streaming"

특정 필드를 검색 할 때도 작동합니다.

  • name:"live"
  • name:"live streaming"

특정 날짜 검색

날짜 종속 검색 필드는 날짜 범위 값을 사용합니다. 날짜 지정을위한 여러 형식이 지원됩니다.

체재 의미
2013-08-01 yyyy-mm-dd 형식의 날짜
2013-08-01T 00:00:00 Z UTC 형식의 시간
-1 일 현재 시간에서 1일을 뺀 값 ( 상대 날짜 참조 )
+1 함 현재 시간과 1일 ( 상대 날짜 참조 )
 

상대 날짜

상대 날짜의 경우 방향 (+ 또는-), 숫자, 기간 (또는 약어)을 지원합니다. 법적 기간은 분 (m), 시간 (h), 일 (d)입니다.

예를 들면 다음과 같습니다.

  • -1day
  • -2days
  • +4hours
  • -60minutes
  • -1d
  • -2d
  • +4h
  • -60m

기간

검색중인 필드가 날짜 인 경우 검색 할 날짜 범위를 지정할 수 있습니다.

  • updated_at:2013-08-01T00:00:00Z..2013-10-08T00:00:00Z

그러면 2013년 8월 1일에서 2013년 10월 8일 사이의updated_at값을 가진 모든 동영상을 검색합니다.

개방형 범위

지정된 시간까지 모든 날짜를 일치 시키거나 지정된 시간 이후의 모든 날짜를 일치 시키려면 범위의 한쪽 끝을 생략하십시오. 예를 들어 지난 2 일 동안 수정 된 모든 비디오를 검색하려면 다음을 수행하십시오.

  • q=updated_at:-2days..

2013년 8월 11일 이전에 수정된 모든 동영상 검색:

  • q=updated_at:..2013-08-11T00:00:00Z

 

특정 용어 포함 및 제외

용어를 필수로 표시할 수 있습니다(반환된 비디오해야하다일치) 또는 금지됨(반환된 동영상은아니다성냥). 이는 용어 바로 앞에+또는-기호로 제어됩니다.

예 의미
+animals 동영상에는' '동물' 이라는 용어가포함되어야 합니다. nameshort description또는long description
-animals 동영상에는' '동물이라는 용어가 포함되어있지 않아야 합니다. nameshort description , 또는long description
+name:animals 동영상에는 이름에' '동물이라는 용어가포함되어야 합니다.
-name:animals 동영상에는 이름에' '동물이라는 용어가포함되어서는 안 됩니다.
