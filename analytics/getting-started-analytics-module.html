Video Cloud Analytics Platform은 Brightcove가 처음부터 빅 데이터 기술의 최신 혁신을 사용하여 대규모로 설계된 실시간에 가까운 최신 분석 플랫폼입니다. 분석 플랫폼은 분석 데이터에 대한 빠르고 실시간에 가까운 액세스를 제공하며 개발자가 Video Cloud Studio에 포함 된 것 외에도 사용자 지정 분석 솔루션 및 보고서를 만들 수 있도록하는 강력한 개방형 API를 포함합니다.

 

온라인 비디오에 대한 특정 목표에 관계없이 청중을 이해하는 것은 성공적인 비디오 전략을 개발하는 데 중요합니다. 시청자를 더 잘 이해할수록 시청자가 쉽게 소비 할 수있는 형식으로 참여할 수있는 콘텐츠를 더 쉽게 만들 수 있습니다. 분석 모듈은 동영상 및 플레이어에 대한 분석을 얻는 데 사용할 수있는 다양한 보고서를 제공합니다.

  • 실적 보고서 -비디오 조회수, 트래픽 소스, 장치 유형 및 장치 운영 체제에 대한 세부 정보를 제공합니다.
  • 참여 보고서 -사용자가 동영상에 참여하는 방식에 대한 세부 정보를 제공합니다. 시청자가 동영상을보고있는 비율
  • 지금 트렌드 - 지난 시간 동안 동시 시청자 수를 제공합니다.
  • 커스텀 -맞춤형 분석 보고서 생성 기능 제공

이러한 보고서를 사용하여 보유한 시청자 수, 시청자가 시청하는 콘텐츠의 양, 동영상 시청을 중단 한시기, 콘텐츠의 트래픽 소스에 대한 최신 보고서를 생성 할 수 있습니다.

이 항목에는 Analytics 모듈 사용 방법을 설명하는 다음 섹션이 포함되어 있습니다.

  • 분석 모듈 시작
  • 시간 범위 선택
  • 비디오 검색
  • 한계

Analytics 모듈에 표시되는 모든 데이터는 Analytics API를 통해서도 사용할 수 있습니다. 자세한 내용은비디오 클라우드 분석 API설명서를 참조하십시오.

분석 모듈 시작

애널리틱스 모듈을 시작하려면

  1. Video Cloud Studio에 로그인합니다.
  2. 클릭하여 분석 모듈을 엽니다. 해석학탐색 헤더에서.

시간 범위 선택

날짜 선택기를 사용하여 보고서의 기간을 지정합니다. 고정 날짜 범위를 선택하거나 사용자 지정 날짜 범위를 선택할 수 있습니다. 하루를 선택하면 그래프가 시간별로 표시됩니다.

보고 시간대

Analytics 모듈의 데이터는 계정의보고 시간대를 사용하여 표시됩니다. 시간대는 달력 컨트롤 아래에 표시됩니다.

보고 시간대는보고 데이터의 일 경계를 계산하는 데 사용됩니다. 보고 시간대 설정을 변경하면 향후 데이터에만 영향을 미치며 변경 사항은 소급 적용되지 않습니다. 이 설정을 변경하면 변경 사항이 적용된 날짜 경계를 볼 때 데이터가 평평하거나 급증 할 수 있습니다. 이 설정에 대한 업데이트는 즉시 적용되지 않을 수 있으며 분석 데이터는 시스템이 변경 사항을 처리 할 때까지 이전 설정을 계속 참조 할 수 있습니다.

보고 시간대는 계정 설정의 일부로 구성 할 수 있습니다. 보고 시간대 변경에 대한 자세한 내용은 계정 설정관리를 참조하십시오 .

비디오 검색

검색 기능을 사용하여 특정 비디오를 검색 할 수 있습니다.

다음 비디오 속성이 검색됩니다.

  • 이름
  • 간단한 설명
  • 장황한 설명
  • 태그
  • 비디오 ID
  • 참조 ID
  • 모든 사용자 정의 필드

검색 필드 아래에 일치하는 목록이 표시됩니다.

비디오 링크를 클릭하면 "상황에 맞는"보고서가 열립니다. 예를 들어 계정 참여를 확인한 다음 검색을 사용하여 특정 동영상을 찾은 경우 동영상 링크를 클릭하면 참여 보고서가 열립니다. 실적 보고서를보고있는 경우 링크를 검색하고 클릭하면 실적 보고서가 열립니다.

애널리틱스 모듈에서 동영상을 검색할 때 사용할 수 있는 전체 옵션 목록은 애널리틱스 모듈에서 동영상을 검색하는방법을 참조하십시오 .

참고: 분석 보고서에는 계정에서 삭제 된 동영상에 대한 분석이 포함됩니다. Analytics 모듈에서 삭제 된 동영상을 검색 할 수 없습니다. 삭제된 동영상에 대한 보고서를 가져오려면 삭제된 동영상의 동영상 ID를 알고 Analytics API를 사용하여 보고서를생성해야 합니다.
