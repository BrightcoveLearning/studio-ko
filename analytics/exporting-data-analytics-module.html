애널리틱스 모듈에 대한 소개는 애널리틱스 모듈시작하기를 참조하십시오 .

보고서 데이터 내보내기

보고서에서 데이터를 내보내려면 다음 단계를 따르세요.

  1. Analytics 모듈로 이동합니다.
  2. 날짜 선택기를 사용하여 기간을 선택합니다.
  3. 보고서를 선택하십시오.
  4. 적절한 링크를 클릭하여 데이터를 CSV또는 XLSX파일로 내보냅니다.
  5. 데이터 파일을 저장하십시오.
