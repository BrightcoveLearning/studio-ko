Facebook은 플레이어 옆이나 아래에 좋아요 버튼을 추가하는 기능을 제공합니다. 좋아요 버튼을 클릭하면 시청자의 Facebook 페이지에 플레이어를 호스팅하는 웹 페이지에 대한 링크가 게시되고 동영상을 좋아 한 사람의 총 수에 한 개인이 추가됩니다. 좋아요 버튼에는 총 좋아요 수를보고하고 동영상을 좋아 한 Facebook 사용자 네트워크에있는 친구의 이름과 프로필 사진을 표시하는 옵션이 있습니다.

Facebook 좋아요 버튼 만들기

Facebook 좋아요 버튼을 만들려면 다음 단계를 따르세요.

  1. 페이스북 개발자 사이트 ( http://developers.facebook.com/docs/reference/plugins/like/) 로 이동합니다 .
  2. 페이지의 지침에 따라 구성 양식에 좋아요 버튼 기본 설정을 입력합니다. 좋아요 URL상자에 플레이어를 호스팅하는 페이지의 URL을 입력합니다.
  3. 코드가져오기를 클릭합니다 .
  4. 딸깍 하는 소리아이프레임대화 상자 상단에서 코드를 복사하십시오.
  5. 플레이어를 호스팅하는 웹 페이지를 열고 플레이어 게시 코드 아래에 코드를 붙여 넣습니다. Facebook 코드 위에<br/>태그를 추가하여 플레이어 아래에 좋아요 버튼을 표시합니다.

페이지가 게시되면 동영상을 좋아 한 시청자 친구의 프로필 사진이 표시됩니다. 여기에있는 이미지 모음은 각 Facebook 사용자마다 다릅니다. 이 옵션을 구현하려면 좋아요 버튼 환경 설정을 구성할때 친구' 얼굴표시를 선택합니다.

이 솔루션은 비디오 수준이 아닌 웹 페이지 수준에서 작동하며 페이지 당 하나의 좋아요 버튼 만 지원합니다. Facebook 사용자 플레이어를 좋아합니다. 플레이어에 재생 목록이 포함 된 경우 좋아요는 개별 동영상을 지정하지 않습니다. 개별 비디오에 대한 상태를 게시하기 위해 사용자는 소셜 공유 기능을 사용하여 비디오를 공유 할 수 있습니다.
