소개

방대한 신규 시장에 진출하고 전자상거래 기회를 크게 확대할 수 있도록 Brightcove는 글로벌 전자상거래 대기업인 Alibaba와의 파트너십을 통해 중국에 비디오를 제공할 수 있는 기능을 도입하게 된 것을 기쁘게 생각합니다.

규정 및 통제로 인해 해외에서 중국으로 비디오를 전송하는 것은 어렵습니다. 역사적으로 우리는 Brightcove 서비스가 국가 내 콘텐츠 전송 네트워크를 통해 비디오 콘텐츠를 배포할 수 있는 솔루션을 제공했습니다. 그러나 Playback API 요청 및 응답, Analytics 비콘을 포함한 필수 통신은 신뢰할 수 없었고 때때로 작동하지 않았습니다.

Brightcove의 China Delivery 솔루션은 이러한 문제를 해결하며 다음 두 부분으로 구성됩니다.

  • 중국 내에서 호스팅되는 프록시 서버
  • 연결된 중국 특정 CDN

프록시는 비 비디오 트래픽 (재생 API 요청, 분석 비콘 및 웹 플레이어 자체)을 중국의 시청자와 외부에있는 Brightcove 인프라로 라우팅합니다.

CDN은 프록시에 연결되어 비디오 및 이미지 콘텐츠를 처리합니다.

설정

Brightcove의 China Delivery 솔루션을 지원하려면 다음 설정이 필요합니다.

  1. 계정 관리자에게 문의하십시오.

    • Brightcove Enterprise 계정이 있는지 확인
    • 중국 배송을 위해 계정을 활성화하십시오. 이것은 Video Cloud의 유료 애드온입니다.
    이 계정은 중국 내외로 배송할 수 있습니다. 별도의 계정은 선택 사항이지만 필수는 아닙니다.

  2. 중국 배달용 플레이어를 구성하십시오. Brightcove 웹 플레이어 또는 기본 SDK 플레이어를 사용할 수 있습니다.

    • Brightcove Player를 통한 중국 배송
    • 네이티브 SDK를 사용한 중국 배송

  3. 유효한 ICP ( 인터넷 콘텐츠 제공자) 라이선스 취득

    • ICP 라이선스에 나열된 승인 된 도메인 또는 하위 도메인 이름
    • 승인 된 도메인에서 호스팅되는 플레이어 게시 용 웹 사이트

한계

다음은 중국 배송의 알려진 제한 사항입니다.

현재 제한 사항

  • 플레이어 구성에는 JSON 편집 및/또는 플레이어 관리 API 사용이 필요합니다. Studio를 통한 "원클릭" 플레이어 구성은 향후를 위한 것입니다.
  • China Delivery를 지원하는 Brightcove Native Android SDK는 2021년 3분기 초에 제공될 예정입니다.

지원되지 않는 기능:

  • Brightcove Live Streaming 및 Cloud Playout은 지원되지 않습니다.
  • 동적 배달 규칙 및 정적 URL은 지원되지 않습니다.

플레이어 및 SDK 플러그인은 정상적으로 사용할 수 있지만 중국 내에서는 외부 서비스에 연결할 수 없으며 Brightcove는 이에 대한 지원을 제공할 수 없습니다. 여기에는 다음이 포함됩니다.

  • DRM
  • 광고 (CSAI + SSAI)
  • 타사 분석 서비스
