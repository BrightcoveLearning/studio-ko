브라이트코브는브라이트코브 비디오 커넥트 워드프레스 플러그인도 제공합니다 . 이 플러그인은 여러 Video Cloud 계정을 지원하며 WordPress 관리 인터페이스 내에서 동영상을 업로드하고, 재생 목록에 추가하고, 동영상으로 단축 코드를 렌더링 할 수 있습니다.

WordPress 페이지에 Brightcove Player를 게시하는 단계

단계 및 화면 캡처는 WordPress 5.2.2를 사용하여 수행되었습니다.

  1. 워드프레스 사이트에 로그인하고 왼쪽 막대의게시물로이동합니다.
  2. 딸깍 하는 소리새로운 걸 더하다새 게시물을 추가합니다.
  3. 게시물 제목을 추가하십시오.
  4. +를 클릭하여 블록을 추가합니다. 서식 지정섹션에서사용자 지정 HTML을 클릭하여 HTML 블록을 추가합니다.
  5. iframe 임베드 코드를 편집기에 붙여넣습니다. 필요에 따라width및height매개변수를 편집합니다.
  6. 미리 보기를클릭하여 플레이어를 미리 봅니다.
  7. 게시... 를 클릭합니다. 제작 설정을 확인한 다음게시를 클릭합니다 .
  8. 사이트에서 플레이어를 보려면 페이지 오른쪽에 있는 게시물보기버튼을 클릭합니다.
  9. 게시 코드 또는 제목을 변경하는 경우업데이트버튼을 클릭하여 변경 사항을 저장해야 합니다.
