Joomla는 인기있는 오픈 소스 콘텐츠 관리 시스템 (CMS)으로 웹 사이트와 강력한 온라인 애플리케이션을 구축 할 수 있습니다. Joomla에 대한 자세한 내용은여기에서 확인할 수있습니다. 이 항목에서는 Brightcove 플레이어를 Joomla 페이지에 쉽게 게시 할 수있는 방법을 보여줍니다.

참고: 이 항목의 단계와 이미지는 Joomla 3.9.6을 사용하여 수행되었습니다.

Brightcove Player를 Joomla 페이지에 게시하려면 다음 단계가 필요합니다.

  • Joomla 편집기 설정 확인
  • 플레이어 삽입에 붙여 넣기 암호

Joomla 편집기 설정 확인

기본적으로 Joomla는 TinyMCE 편집기를 사용하도록 구성됩니다. TinyMCE 편집기는 편집기에 붙여 넣을 때 특정 HTML 태그를 제거하도록 구성됩니다. 표준 (iframe) 플레이어 게시 코드를 Joomla 편집기에 붙여 넣으려면 편집기에서<iframe>태그를 제거하지 않도록해야합니다.

  1. Joomla 백엔드에 관리자로 로그인합니다.
  2. 확장 기능 > 플러그인을 선택합니다 .
  3. 검색 상자에 tinymce를 입력하고 검색 버튼을 클릭합니다.
  4. 하나의 항목이 반환되었는지 확인합니다. 에디터 - TinyMCE입니다.
  5. 에디터 - TinyMCE플러그인 링크를 클릭하여 설정을 편집합니다.
  6. 금지된 요소섹션을 찾아 iframe이 목록에 없는지 확인합니다. iframe이 목록에 있으면 목록에서 제거합니다.
    참고: Joomla는금지된 요소필드에 값이 있어야 합니다. iframe항목을 제거하면 이 필드를 비워 둘 경우 이필드에“none”을 입력합니다.
  7. 딸깍 하는 소리구하다페이지 상단에서 변경 사항을 저장합니다.

TinyMCE 편집기는 고급 (페이지 내) 플레이어 퍼블리싱 코드를 사용할 때<script>태그를 제거합니다. 이 문제를 해결하는 한 가지 방법은 TinyMCE 편집기를 사용하지 않도록 Joomla 인스턴스를 구성하는 것입니다. 다른 편집기를 선택하려면 다음 단계를 따르십시오.

  1.  시스템  >  글로벌 구성으로 이동합니다 .
  2. 클릭 대지 탭.
  3. 찾기 기본 편집기 선택권.
  4. 이 옵션을 편집기 - 없음으로 설정합니다 . 에디터 - CodemIRror를 선택하면<script>태그가 제거되지 않습니다.
  5. 저장 및 닫기를 클릭합니다 .

Joomla 설치가 다른 편집기를 사용하도록 구성된 경우 Joomla에 코드를 붙여넣고 저장할 때 편집기가 플레이어 게시 코드를 편집하지 않았는지 확인해야합니다. 이 경우 다른 Joomla 설정을 변경하거나 확장 프로그램 ( Jumi또는 Sourcerer ) 을 사용하여 JavaScript 코드를 편집기에 붙여 넣을 수 있습니다.

플레이어 소스 코드에 붙여 넣기

코드를 편집하지 않도록 편집기가 구성되면 플레이어 소스 코드를 생성 할 수 있습니다.

  1. 미디어 모듈을 사용하여 게시하려는 비디오의 소스 코드를 생성하십시오.
  2. Joomla에서 편집하려는 페이지를 엽니 다.
  3. 편집기 하단의 편집기전환버튼을 클릭하여 HTML 모드로 전환합니다.
  4. 플레이어 퍼블리싱 코드에 붙여넣고저장 및 닫기를 클릭합니다 .
  5. 동영상이 페이지에 나타나고 재생되는지 확인합니다.
