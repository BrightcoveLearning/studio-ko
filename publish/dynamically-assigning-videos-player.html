플레이어와 관련된 비디오를 변경해야하는 경우가있을 수 있습니다. 예를 들어, 시청자가보고 싶은 비디오를 제어 할 수있는 능력이있을 수 있습니다. 플레이어의 동영상은 시청자가 사이트의 어느 부분에서 왔는지 기반으로 할 수 있습니다.

플레이어에 다른 비디오를 표시해야하는 경우 하나의 플레이어를 사용한 다음 플레이어와 연결된 비디오를 동적으로 변경하는 것이 좋습니다.

이 항목에서는 다음 기술에 대해 설명합니다.

  • URL 매개 변수로 동영상 ID 전달
  • 참조 ID를 URL 매개 변수로 전달
  • 임베드 수정 동영상 ID를 제공하는 코드
  • 임베드 수정 참조 ID를 제공하는 코드
참고: 플레이어에 정적으로 바인딩 된 비디오를 변경할 수 있지만 이는 좋은 방법이 아닙니다. 이렇게하면 동적으로 추가 된 비디오가 표시되기 전에 정적으로 바인딩 된 비디오의 포스터 이미지가 플레이어에 잠시 나타날 수 있습니다. 따라서 비디오가 동적으로 변경되는 상황에서 사용하기 위해 처음에 연결된 비디오가 없는 플레이어를 만드는 것이 가장 좋습니다.

Catalog API를 사용하여 콘텐츠를 플레이어에 동적으로 할당 할 수도 있습니다. 자세한 내용과 샘플 코드는 개발자 문서 소스 비디오동적으로 변경 을 참조하십시오 .

URL 매개 변수로 동영상 ID 전달

양식을 사용하여 비디오 ID를 URL 매개 변수로 전달하여 플레이어의 비디오 소스를 변경할 수?videoId=YOUR_VIDEO_ID있습니다.

http://players.brightcove.net/20318290001/f1d70e07-8480-4bbd-b4ef-747e9333a034_default/index.html?videoId=4292158104001

참조 ID를 URL 매개 변수로 전달

양식을 사용하여 참조 ID를 URL 매개 변수로 전달하여 플레이어의 비디오 소스를 변경할 수?videoId=ref:YOUR_VIDEO_REF_ID있습니다.

http://players.brightcove.net/20318290001/f1d70e07-8480-4bbd-b4ef-747e9333a034_default/index.html?videoId=ref:1234

비디오 ID를 제공하도록 소스 코드 수정

비디오 ID를 사용하여 Video Cloud 비디오를 플레이어 내장 코드에 직접 할당 할 수 있습니다.

인 페이지 임베드 코드

data-video-id="YOUR_VIDEO_ID"<video>태그에 속성을 추가합니다.

<video-js   data-account="57838016001"
  data-player="6e44f16e-31e9-48b2-83b9-6edd71217540"
  data-embed="default"
  data-video-id="68143720001"
  class="video-js"
  controls></video-js>
<script src="https://players.brightcove.net/57838016001/6e44f16e-31e9-48b2-83b9-6edd71217540_default/index.min.js"></script>
참고: 플레이어 자바 스크립트가 실행되면 시작할 때만 data-video-id 속성에서 동영상 ID를 읽습니다. 예를 들어 JavaScript의 setAttribute () 메서드를 사용하여 값을 변경해도 플레이어가 비디오를로드하지 않습니다. JavaScript를 사용하여 비디오를 변경하는 방법에 대한 예제는플레이어 카탈로그개발자 문서를 참조하십시오.

iframe 삽입 코드

추가하다?videoId=YOUR_VIDEO_ID URL의 끝까지src iframe의 속성:

<iframe src='//players.brightcove.net/57838016001/6e44f16e-31e9-48b2-83b9-6edd71217540_default/index.html?videoId=68143720001'
 allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>

참조 ID를 제공하도록 소스 코드 수정

참조 ID를 사용하여 Video Cloud 비디오를 플레이어 내장 코드에 직접 할당 할 수 있습니다.

참고: 특수 문자가 있는 참조 ID를 사용하는 경우 게시 코드에서 특수 문자를 이스케이프 처리해야 합니다. 그렇지 않으면 문제가 발생할 수 있습니다.

인 페이지 임베드 코드

data-video-id="ref:YOUR_VIDEO_REF_ID"<video>태그에 속성을 추가합니다.

<video-js data-account="57838016001"
  data-player="6e44f16e-31e9-48b2-83b9-6edd71217540"
  data-embed="default"
  data-video-id="ref:68143720001"
  class="video-js"
  controls></video-js>
<script src="https://players.brightcove.net/57838016001/6e44f16e-31e9-48b2-83b9-6edd71217540_default/index.min.js"></script>
