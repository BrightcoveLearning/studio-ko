참조 ID를 사용하는 이유는 무엇입니까?

가능하면 동영상 ID 또는 재생 목록 ID 대신 참조 ID를 사용하는 것이 좋습니다. 몇 가지 장점은 다음과 같습니다.

  • 실수로 비디오 또는 재생 목록을 삭제 한 경우 모든 플레이어를 업데이트 할 필요가 없습니다. 대신 다른 동영상을 업로드하거나 재생 목록을 다시 만들고 삭제 된 항목과 동일한 참조 ID를 할당하면됩니다.
  • 콘텐츠 관리 시스템 (CMS)으로 동영상과 웹 페이지를 관리 할 때 참조 ID를 콘텐츠 ID와 쉽게 일치시킬 수 있습니다.

개요

기본적으로 Video Cloud Studio에서 플레이어로 비디오를 게시하면 비디오 ID를 사용한다는 것을 알 수 있습니다. 참조 ID를 사용하려면 참조 식별자 앞에 붙이기만 하면ref:됩니다.

따라서 다음과 같은 비디오 ID를 사용하는 대신:

<video-js data-video-id="4746402489001"

다음과 같은 참조 ID를 사용할 수 있습니다.

<video-js data-video-id="ref:skivideo"

재생 목록 ID에도 동일하게 적용됩니다. 자세한 내용은 아래 예를 참조하십시오.

참고: 참조 ID는 계정의 모든 동영상에서고유해야 합니다.

참조 ID의 고유성을 보장하기 위해 참조 ID를 포함하는 작업은 최대 30 초 동안 ID를 잠급니다. 즉, 동영상에서 참조 ID를 제거하거나 해당 동영상을 삭제 한 다음 즉시 다른 동영상에 참조 ID를 사용하려고하면 409 오류와 함께 작업이 실패합니다. 이 오류가 발생한 후 작업을 다시 시도하기 전에 30 초 이상 기다리십시오.

또한 참조 ID를 사용하여 동영상을 만들려고했는데 시도가 실패하면 (참조 ID와 관련이없는 이유로) 해당 참조 ID가 30 초 동안 잠기고 동영상 만들기를 반복 할 때까지 실패합니다. 참조 ID가 잠금 해제 될 때까지 충분히 기다립니다.

참조 ID 이름으로 이스케이프해야 할 문자는 HTML 사양을 따르는 큰따옴표와 큰 따옴표로 묶인 속성의 앰퍼샌드뿐입니다 .

참조 ID 추가

먼저 동영상에 참조 ID를 추가하세요.

  1. Studio에서미디어모듈을 열고 비디오 제목을 클릭하여 비디오 속성을 엽니다.
  2. 비디오 정보섹션을 찾아편집을 클릭합니다 .
  3. 참조 ID속성의 경우 참조 ID 값을 입력합니다.
  4. 변경 사항을 저장하십시오.

비디오 ID 교체

동영상 ID는 참조 ID로 대체 할 수 있습니다.

iframe

iframe 구현을 사용하려면 다음을 수행합니다.

  1. 미디어모듈에서 플레이어와 함께 비디오를 게시합니다.
  2. 표준포함 코드를 복사하여 HTML 페이지 코드에 붙여 넣습니다.
  3. videoId값을 ref:용어 앞에 붙인 적절한 참조 ID로 변경합니다.
<!doctype html>
<html>

<head>
    <meta charset="UTF-8">
    <title>Video Reference ID</title>
</head>

<body>

  <iframe src="https://players.brightcove.net/1752604059001/default_default/index.html?videoId=ref:skivideo&muted"
      allowfullscreen
      webkitallowfullscreen
      mozallowfullscreen
      width="640"
      height="360"></iframe>

</body>
</html>

인 페이지 삽입

인 페이지 삽입 구현을 사용하려면 다음을 수행하십시오.

  1. 미디어모듈에서 플레이어와 함께 비디오를 게시합니다.
  2. 고급포함 코드를 복사하여 HTML 페이지 코드에 붙여 넣습니다.
  3. data-video-id값을 ref:용어 앞에 붙인 적절한 참조 ID로 변경합니다.
<!doctype html>
<html>

<head>
    <meta charset="UTF-8">
    <title>Video Reference ID</title>
</head>

<body>

  <video-js data-video-id="ref:skivideo"
      data-account="1752604059001"
      data-player="default"
      data-embed="default"
      data-application-id
      class="video-js"
      controls muted
      width="640"
      height="360"></video-js>
  <script src="https://players.brightcove.net/1752604059001/default_default/index.min.js"></script>

</body>
</html>

재생 목록 ID 바꾸기

재생 목록 ID는 참조 ID로 바꿀 수 있습니다.

iframe

iframe 구현을 사용하려면 다음을 수행합니다.

  1. 미디어모듈에서 플레이어와 함께 비디오를 게시합니다.
  2. 표준포함 코드를 복사하여 HTML 페이지 코드에 붙여 넣습니다.
  3. playlistId값을 ref:용어 앞에 붙인 적절한 참조 ID로 변경합니다.
<!doctype html>
<html>

<head>
    <meta charset="UTF-8">
    <title>Video Reference ID</title>
</head>

<body>

  <iframe src="https://players.brightcove.net/1752604059001/H1EyAYN3_default/index.html?playlistId=ref:allTimePlaylist"
      allowfullscreen
      webkitallowfullscreen
      mozallowfullscreen
      width="640"
      height="360"></iframe>

</body>
</html>

인 페이지 삽입

인 페이지 삽입 구현을 사용하려면 다음을 수행하십시오.

  1. 미디어모듈에서 플레이어와 함께 비디오를 게시합니다.
  2. 고급포함 코드를 복사하여 HTML 페이지 코드에 붙여 넣습니다.
  3. data-playlist-id값을 ref:용어 앞에 붙인 적절한 참조 ID로 변경합니다.
<!doctype html>
<html>

<head>
  <meta charset="UTF-8">
  <title>Video Reference ID</title>
  <style type="text/css">
    .video-js {
      float: left;
    }
    .vjs-playlist {
      width: 280px;
      height: 360px;
      overflow-x: hidden;
    }
  </style>
</head>

<body>

  <video-js data-playlist-id="ref:allTimePlaylist"
      data-account="1752604059001"
      data-player="H1EyAYN3"
      data-embed="default"
      data-application-id
      class="video-js"
      controls
      width="640"
      height="360"></video>
  <script src="https://players.brightcove.net/1752604059001/H1EyAYN3_default/index.min.js"></script><div class="vjs-playlist"></div>

</body>
</html>
