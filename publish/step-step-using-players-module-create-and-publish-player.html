참고: 재생 목록 또는 단일 비디오를 플레이어에 게시하려는 경우 미디어 모듈 내의 게시 기능을 사용하십시오. 자세한 내용은 미디어 모듈을 사용하여 비디오 및 재생 목록게시를 참조하십시오 .

청중

플레이어 모듈을 사용하여 동영상을 만들고 게시해야하는 모든 동영상 게시자.

전제 조건

  • 플레이어 모듈에 대한 액세스 권한이있는 Video Cloud 계정
  • 게시할 Video Cloud 계정의 동영상; 이 빠른 시작에 사용된 비디오는여기에서다운로드하거나 자신의 콘텐츠를 사용할 수 있습니다.
  • 이 Quick Start와 함께 제공된 비디오를 사용하는 경우 Video Cloud 계정에 업로드하십시오.

명령

Video Cloud Studio를 사용하여 Brightcove 플레이어를 만들고 게시하려면 다음 작업을 완료해야합니다.

  • 플레이어 만들기
  • 플레이어 크기 구성
  • 플레이어 게시 및 미리보기
  • 비디오 콘텐츠를 플레이어에 동적으로 할당
  • 스타일 속성 구성
  • 소셜 공유 속성 구성
  • 플레이어 미리보기
  • 최종 플레이어 게시

플레이어 만들기

새 플레이어를 만들려면 다음 단계를 따르십시오.

  1. 비디오 클라우드 스튜디오에 로그인합니다.
  2. 현재 Video Cloud 계정 이름이 페이지의 오른쪽 상단에 나타납니다. 계정이 여러 개인 경우현재 사용자 - 현재 계정링크를 클릭하고 플레이어를 만들려는 계정을 선택합니다.
  3. 탐색 헤더에서플레이어링크를 클릭합니다.
  4. [ 새 플레이어]버튼을 클릭합니다.
  5. 플레이어의이름과간단한 설명을입력합니다. 이 퀵 스타트의 경우 플레이어 이름은샘플 플레이어가 됩니다 .
  6. 플레이어가 플레이어 목록에 나타나는지 확인합니다.

플레이어 모듈을 사용하여 새 플레이어를 만들면 플레이어는 Brightcove에서 게시 한 최신 플레이어 버전을 사용하여 빌드됩니다. 플레이어 버전을 보려면 플레이어 모듈을 열고 확인하려는 플레이어를 클릭하십시오. 플레이어 정보 프로퍼티에는게시된 플레이어 버전이 표시됩니다 .

모든 플레이어가 동일한 버전이 아닐 수 있습니다. Brightcove가 Brightcove 플레이어를 계속 개발하고 개선함에 따라 새로운 플레이어 버전이 출시 될 것입니다. 기본적으로 업데이트는 모든 플레이어에게 자동으로 적용됩니다. 플레이어의 자동 업데이트를 방지하려면업데이트 모드설정을수동 (플레이어 잠금이라고도 함) 으로 설정해야 합니다. 자세한 내용은 플레이어 업데이트 모드 및 버전설정을 참조하십시오 .

플레이어 크기 구성

플레이어 크기를 추가하면 너비 및 높이 매개 변수가 iframe 소스 코드에 추가됩니다. 치수를 비워 두면, 플레이어는 페이지의 어떤 크기로든 넣을 수 있으며 공간을 채우도록 확장됩니다. <div> 플레이어 크기를 설정하려면 다음 단계를 따르십시오.

  1. 샘플 플레이어 링크를클릭하여 플레이어 속성 페이지를 엽니다.
  2. 왼쪽 탐색 메뉴에서플레이어 정보를 클릭합니다.
  3. 설정종횡비에게 16:9 .
  4. 설정플레이어 너비에게 640 .
  5. 저장을 클릭합니다 .

플레이어 정보 프로퍼티에 대한 자세한 내용은 플레이어 정보 속성설정을 참조하십시오 .

플레이어 게시 및 미리보기

플레이어를 게시하면 변경 사항이 게시 된 플레이어로 푸시됩니다. 미리 보기 플레이어, 게시된 플레이어 및 플레이어 게시 코드에 대한 자세한 내용은 다음을 참조하세요. 플레이어 임베드 생성 암호 .

플레이어를 게시하려면 다음 단계를 따르십시오.

  1. 게시및 포함을 클릭한 다음 변경 사항게시를 클릭하여 플레이어를 게시합니다.
  2. 게시가 성공했는지 확인합니다.
  3. 플레이어 URL을클릭하여 새 브라우저 탭에서 미리 보기 플레이어를 엽니다. 플레이어에 할당 된 비디오가 없으므로 플레이어가 비어 있습니다.

비디오 콘텐츠를 플레이어에 동적으로 할당

이 Quick Start에서는 URL 매개 변수를 사용하여 Video Cloud 비디오 ID를 플레이어에 동적으로 할당합니다. 이렇게하려면 다음 단계를 따르십시오.

  1. 미디어 모듈을 열고 게시 할 비디오를 찾습니다. 동영상 ID는 동영상 이름 아래에 표시됩니다.
  2. 비디오 ID를 클립 보드에 복사합니다.
  3. 미리보기 플레이어가있는 탭으로 돌아갑니다. 표시할 동영상 ID를 동적으로 전달하려면 URL에 다음을 추가하고 Enter 키를 누릅니다.
    ?videoId=your video id
    메모: URL 매개 변수 이름은 대/소문자를 구분합니다.

    전체 URL은 다음과 같습니다.

    https://players.brightcove.net/1486906377/Ctzra7vEL_default/index.html?videoId=5798846643001
  4. 비디오가 나타나는지 확인하십시오.

스타일 속성 구성

스타일 속성은 플레이어의 모양을 사용자 지정하는 데 사용됩니다. 플레이어 스타일링에 대한 자세한 내용은 플레이어스타일 지정을 참조하십시오 .

스타일 설정을 구성하려면 플레이어 모듈로 돌아가 다음 단계를 따르십시오.

  1. 샘플 플레이어 링크를클릭하여 플레이어 속성 페이지를 엽니다.
  2. 딸깍 하는 소리스타일링왼쪽 탐색 메뉴에서
  3. 재생 버튼모양과재생 버튼 위치를 선택합니다 .
  4. [ 색상 테마]섹션에서 [ 재생 버튼]색상 피커를 클릭합니다.
  5. 제어 장치의 탑승면에있는 빨간색을 클릭하십시오. 사각형을 클릭하여 빨간색 음영을 선택합니다.
  6. 컨트롤 외부를 클릭하여 색상 선택기를 축소합니다.
  7. 미리보기 플레이어 아래에는 플레이어가 변경되었음을 알리는 메시지가 있습니다.

소셜 공유 속성 구성

소셜 공유가 활성화되면 플레이어 하단의 플레이 헤드 옆에 공유 아이콘이 나타납니다. 이를 통해 시청자는 인기있는 소셜 미디어 사이트에 귀하의 콘텐츠를 공유 할 수 있습니다. 플레이어의 소셜 속성을 구성하는 방법에 대한 자세한 내용은 플레이어의 소셜 속성구성을 참조하십시오 .

소셜 설정을 구성하려면 다음 단계를 따르십시오.

  1. 왼쪽 탐색 메뉴에서 Social을 클릭합니다.
  2. 공유사용확인란을 선택합니다.
  3. 모든 소셜 미디어 사이트를 선택하십시오.

플레이어 미리보기

페이지의 미리보기 플레이어는 플레이어의 스타일 및 소셜 변경 사항을 반영해야합니다.

참고: 이전에 열었던 게시 된 플레이어로 돌아가 페이지를 새로 고치면 이러한 변경 사항이 게시 된 플레이어에 적용되지 않음을 알 수 있습니다. 색상 및 소셜 변경 사항은 플레이어 모듈 내부에 변경 사항이 게시될 때까지 게시된 플레이어에게 푸시되지 않습니다.

미리보기 플레이어에 대한 게시 코드를 생성 할 수도 있습니다. 예를 들어 마케팅 부서에서 플레이어의 모든 변경 사항을 승인해야 할 수 있습니다. 변경 사항을 승인 할 수 있도록 미리보기 플레이어에 URL을 보낼 수 있습니다. 미리보기 플레이어의 게시 코드를 생성하려면 다음 단계를 따르세요.

  1. 플레이어 모듈 홈페이지에서코드 및 URL 삽입클릭플레이어 미리보기 . 플레이어 임베드 코드 미리보기 대화 상자가 표시됩니다.
  2. 플레이어 URL을클릭하여 새 브라우저 탭에서 미리 보기 플레이어를 엽니다.
  3. 미리보기 플레이어는 브라우저 페이지의 전체 너비를 차지해야하며 미리보기 플레이어임을 나타내는 Brightcove 로고 오버레이가 있어야합니다. 색상 변경 및 바이러스 프로모션 변경이 보이는지 확인합니다.
  4. 미리보기 플레이어를 열어두고 플레이어 모듈로 돌아갑니다.

최종 플레이어 게시

미리보기 플레이어의 변경 사항은 변경 사항이 게시 될 때까지 게시 된 플레이어에 표시되지 않습니다. 변경 사항을 게시하려면 다음 단계를 따르십시오.

  1. 플레이어 모듈에서게시및 임베드를 클릭한 다음 변경 사항게시를 클릭하여 플레이어를 게시합니다.
  2. 게시가 성공했는지 확인합니다.
  3. 게시 된 플레이어가있는 브라우저 탭으로 돌아갑니다. 미리보기 플레이어는 Brightcove 로고 오버레이를 표시합니다.
  4. 페이지를 새로 고침하고 이제 게시 된 플레이어에 색상 변경 및 바이러스 프로모션 설정이 표시되는지 확인합니다. 게시 된 플레이어에 변경 사항이 표시되기까지 약간의 지연이있을 수 있습니다.
