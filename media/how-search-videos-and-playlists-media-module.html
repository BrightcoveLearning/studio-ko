비디오 검색

술어

이 섹션에서정확한 용어는 검색 문자열이 따옴표로 묶여있음을 의미합니다. 예를 들면 다음과 같습니다.

여러 비디오 ID를 검색하려면 id:xxx id:xxx id:xxx id:xxx 구문을 사용하십시오 . 예를 들어, id:5985323381001 id:5715142164001 .

다음 비디오 필드가 검색됩니다.

  • 비디오 ID ( id )
  • 이름 ( name )
  • 간단한 설명 ( description )
  • 자세한 설명 ( long_description) )
  • 태그 ( tags )
  • 참조 id ( reference_id )
  • 사용자 지정 필드 ( custom_fields또는{field_name} )

비디오 필드에 대한 쿼리 지원은 다릅니다. 일부 필드는 부분 단어 검색 및형태소 분석 검색을지원하며 일부 필드는 지원하지 않습니다.

형태소 분석이란 무엇입니까?

검색어의 어간이 공통 인 형태소 반환 단어를 지원하는 비디오 필드입니다. 또한 형태소 분석은 부분 단어가 아닌 전체 단어 입력 만 지원합니다.

  • 예 1: 검색 시 다음을 포함하는 결과가running반환됩니다. running , run , runs
  • 예 2: 검색 시vid다음을 포함하는 결과는 반환되지 않습니다. video

일반 규칙

  • 정확히 일치하려면 문자열을 따옴표로 묶으십시오.
  • 기본적으로 Video Cloud는 단어 사이에 OR 논리 연산자를 가정합니다.
  • 검색은 대소 문자를 구분하지 않습니다.

검색어 보고

태그

고급 검색을 사용하여 비디오 태그를 기반으로 비디오를 반환 할 수 있습니다. 태그를 사용하여 검색 할 때 세 가지 옵션이 있습니다.

  • 어떤 -태그 (OR)가있는 동영상 반환
  • 모두 -반환 동영상은 모든 태그 (AND)가됩니다.
  • 없음 -이러한 태그가없는 동영상 반환

고급 검색 구문

고급 검색 대화 상자를 사용하여 비디오를 검색하면 검색 기준을 입력 한 후 검색 상자에 고급 검색 구문이 나타납니다. 이 섹션에서는 검색 구문에 대해 설명합니다. 고급 검색 대화 상자를 사용하지 않고 검색 구문을 검색 상자에 입력 할 수도 있습니다. 고급 검색 대화 상자를 열려면추가 검색 옵션링크를 클릭합니다.

검색어를 필수로 표시하려면(반환된 비디오 ~해야하다일치) 또는 제외(반환된 동영상) 해서는 안 된다일치), 추가 +또는 -용어 바로 앞의 기호.

다음 검색 규정 자 및 값이 지원됩니다.

예선 유효한 값
name 문자열 또는 인용 된 문자열

tags 문자열 또는 인용 된 문자열

< custom_field_name > - 지정된 사용자 정의 필드 이름을 검색합니다. 문자열 또는 인용 된 문자열
state[1] ACTIVE , INACTIVE , PENDING
updated_at 날짜 범위

메모

[1] 계정에 공유되었지만 승인되지 않은 동영상은 PENDING 상태가 됩니다.

[2] 비디오가 마지막으로 활성화된 날짜입니다.

[3] 비디오에 재생 가능한 변환이 하나 이상 있으면 비디오의 완전한 상태가 true입니다. 모든 트랜스 코딩이 완료되었음을 의미하는 것은 아닙니다.

  • -2days

  • -60minutes

  • -2d
  • +4h
  • -60m

검색 저장

자주 실행하는 검색을 저장할 수 있으므로 검색 구문을 다시 입력하지 않고도 빠르게 다시 실행할 수 있습니다. 보려는 비디오를 반환하는 검색 구문을 입력한 후 검색저장을 클릭합니다 . 저장된 검색의 이름을 입력한 다음저장을 클릭합니다 .

저장된 검색을 다시 실행하려면 검색 상자 아래에 있는저장된 검색링크를 클릭하고 저장된 검색을 클릭하여 실행합니다.

저장된 검색을 삭제하려면 삭제 아이콘() 검색 옆에 있습니다.
