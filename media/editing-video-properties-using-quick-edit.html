빠른 편집 기능은 여러 비디오의 비디오 속성을 편집하는 편리한 방법을 제공합니다. 확인란을 클릭 한 다음 다른 확인란을 Shift- 클릭하여 여러 비디오를 선택할 수 있습니다 (Shift 키를 누른 상태에서 두 번째 비디오를 클릭하면 두 비디오와 그 사이의 모든 비디오가 선택됩니다). 빠른 편집을클릭하면 미디어 모듈의 오른쪽에 있는 빠른 편집 패널이 열립니다.

빠른 편집 패널에는 비디오 속성의 하위 집합이 표시됩니다.

  • 이름
  • 간단한 설명
  • 장황한 설명
  • 추가 할 태그
  • 제거 할 태그
  • 관련 링크 URL
  • 관련 링크 텍스트
  • 광고하는
  • 광고 키
  • 시작 / 종료 날짜 예약
  • 지오 필터링
  • 사용자 정의 필드
  • 재생 제한
참고: 이러한 모든 속성에 대한 설명은 미디어 모듈을 사용하여 비디오 속성편집문서를 참조하십시오.

빠른 편집 패널을 사용하여 값을 지정하십시오. 완료되면 저장을 클릭하여 변경 사항을 저장하거나 취소를 클릭하여 종료합니다.

빠른 편집 사용에 대한 참고 사항

  • 하나 이상의 비디오에 속성에 할당된 값이 있는 경우 혼합값편집이 표시됩니다.
  • 속성 이름 뒤의 별표 (*)는 필수 속성을 나타냅니다.
  • 등록 정보 변경 사항을 취소하려면복원을 클릭합니다.
