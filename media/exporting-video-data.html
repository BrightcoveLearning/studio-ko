미디어 모듈은 모든 비디오 또는 비디오 하위 집합에 대한 비디오 메타 데이터를 내보내는 기능을 제공합니다. 이 기능을 사용할 때 다음 사항에 유의하십시오.

  • Brightcove는 한 번에 15K 미만의 비디오를 내보낼 것을 권장합니다. 15K 이상의 비디오를 내 보내야하는 경우 검색 기능을 사용하여 비디오 하위 집합을 반환하고 한 번에 여러 "배치"를 내 보냅니다.
  • 변환 및 이미지의 URL은 고정되지 않습니다. Brightcove는 수시로 미디어 자산의 저장소를 재구성하며 이러한 상황이 발생하면 특정 자산의 URL이 변경됩니다. 이러한 자산에 대해 하드 코딩 된 URL을 사용하는 경우 링크가 어느 시점에서 끊어 질 수 있습니다. 미디어 링크가 끊어지지 않도록 하는 가장 좋은 방법은 런타임에 CMS API를 사용하여 Video Cloud에서 미디어를 검색하는것입니다.
  • Excel에서 열면 한국어 및 일본어와 같은 더블바이트 문자가 제대로 표시되지 않습니다. 이는 생성된.csv 파일이 UTF-8 에 의해 인코딩되고 Excel이 제대로 지원하지 않기 때문에 발생합니다. 이 문제를 해결하려면 ANSI에 의해.csv 파일을 다시 인코딩한 다음 Excel에서 올바르게 표시됩니다.

비디오 메타 데이터를 내보내려면 다음 단계를 따르십시오.

  1. 열기미디어기준 치수.
  2. 특정 비디오에 대한 데이터를 내보내려면 비디오 옆의 확인란을 선택하십시오.
  3. 딸깍 하는 소리내 보내다페이지 상단에 있습니다.
  4. 내보낼 비디오를 선택하십시오.
    • 이 목록의 모든 비디오 -계정의 모든 비디오에 대한 비디오 메타 데이터 내보내기
    • x 선택한 동영상 -선택한 비디오에 대해서만 비디오 메타 데이터 내보내기
  5. 비디오 변환 URL을포함하려면 비디오 변환의 현재 URL 포함을 선택합니다. 많은 수의 비디오를 선택한 상태에서이 옵션을 선택하면 내보내기 파일을 생성하는 데 필요한 시간이 늘어납니다.
    참고: 이 옵션을 선택하면 최대 1000개의 비디오에 대한 비디오 메타데이터를 내보낼 수 있습니다.
  6. 데이터내보내기를 클릭합니다 . 데이터 내보내기가 시작됩니다.
  7. 내보내기 파일이 준비되면 파일다운로드를 클릭하여 비디오 데이터 파일을 로컬 컴퓨터로 다운로드합니다.

.csv 파일은 Video Cloud 계정 번호로 이름이 지정됩니다. 타사 스프레드 시트 프로그램을 사용하여 내 보낸 비디오 데이터를 볼 수 있습니다.
