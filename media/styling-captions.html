캡션의 기본 모양과 느낌은 변경 될 수 있으므로 캡션의 스타일을 지정할 수 있습니다. 캡션 스타일을 지정할 수있는 몇 가지 방법이 있습니다.

  • WebVTT 파일에 스타일 지정 옵션을 추가하여
  • CSS 파일에서 사용자 정의 클래스를 생성하여
  • CSS 스타일 가이드
  • 사용자 스타일링 옵션
참고: Safari는 기기에서 네이티브 캡션을 사용하며 기기 설정이 vtt 스타일보다 우선합니다.

WebVTT 파일에 스타일 옵션 추가

스타일 옵션은 WebVTT 파일에 직접 적용하여 캡션과 텍스트의 위치와 방향을 변경할 수 있습니다. 다음은 몇 가지 예입니다.

  • 굵은 텍스트 
    <b>Lorem ipsum</b>
  • 기울임 꼴 텍스트 
    <i>dolor sit amet</i>
  • 밑줄이 그어진 텍스트 
    <u>consectetuer adipiscing</u>

Brightcove Player는 캡션의 위치를 관리하므로 스타일을 사용하여 캡션의 위치 나 방향을 변경할 수 없습니다. 인라인 스타일은 많은 스타일 옵션을 제공하지 않습니다. 더 다양한 스타일 옵션의 경우 다음 섹션에 설명 된대로 CSS 파일을 사용할 수 있습니다.

인라인 스타일에 대한 자세한 내용은이 링크를 참조하십시오 .

CSS 파일에서 사용자 정의 클래스 만들기

사용자 정의 클래스는 CSS 파일의 일부로 생성 된 다음 WebVTT 파일의 요소에 적용 할 수 있습니다. 예를 들어 아래 클래스는 텍스트 색상을 흰색으로, 배경색을 파란색으로 변경하고 글꼴을 굵게 표시합니다.


<style>
   .captionstyle {
      color:white;
      background-color:blue;
      font-weight:bold
    }
</style>

그런 다음 WebVTT 파일을 수정하여 새captionstyle선택기를 적용할 수 있습니다. c접두사는 클래스 선택기 (WebVTT 형식 표준입니다) 사용되는 스타일을 나타내는 데 사용됩니다.

WEBVTT 00:00:00.000 --> 00:00:03.530 정렬:중간 선:84% 00:00:03.530 --> 00:00:06.510 정렬:중간 선:90% <c.captionstyle>이 비디오에서는 에 로그인하고 탐색하는 방법을 배웁니다. </c> 00:00:06.510 --> 00:00:09.550 정렬:중간 선:84% <c.captionstyle>비디오 클라우드 스튜디오 내부. </c>

캡션 파일의 스타일 지정 옵션에 대한 자세한 내용은 W3C WebVTT 사양 문서를참조하거나이 링크를 참조하십시오 .

캡션의 사용자 스타일링

iOS 기기 사용자의 경우 캡션 스타일은 iOS 설정을 통해 제어됩니다. 설정에 액세스하려면 다음을 선택하십시오. 설정 > 일반적인 > 접근성 > 자막 및 캡션 > 스타일 . 여기에서 미리 정의 된 스타일을 선택하거나 새 스타일을 만들 수 있습니다.

데스크톱 및 Android 장치의 뷰어는 캡션 설정 메뉴를 사용하여 캡션의 모양을 변경할 수 있습니다.

캡션 설정 메뉴를 클릭하면 다음 캡션 속성을 설정할 수있는 오버레이가 표시됩니다.

  • 전경색 및 불투명도
  • 배경색 및 불투명도
  • 창 색상 및 불투명도
  • 글꼴 크기
  • 텍스트 가장자리 스타일
  • 글꼴 군

기본값을클릭하여 캡션을 기본 스타일링으로 되돌립니다.
