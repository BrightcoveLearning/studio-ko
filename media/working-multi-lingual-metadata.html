비디오 속성을보고 편집하려면 미디어 모듈을 열고 비디오 제목 링크를 클릭합니다. 비디오 속성 페이지가 열립니다.

참고: 다국어 메타데이터를 비디오와 연결하려면 언어 설정의 일부로 기본 언어와 하나 이상의 추가언어를 구성해야합니다.

비디오 속성 페이지가 열리면 비디오 메타 데이터가 기본 언어를 사용하여 표시됩니다. 언어 선택기가 비디오 정보 섹션 위에 표시되고 기본 언어가 표시됩니다. 다른 언어를 선택하려면 버튼을 클릭하십시오. 이 예에서는 영어가 기본 언어로 구성되고 프랑스어와 스페인어가 추가 언어로 추가되었습니다.

다국어 메타 데이터 추가

비디오에 다국어 메타 데이터를 추가하려면 언어 선택기를 사용하여 메타 데이터를 추가 할 언어를 선택합니다.

비디오 속성이 업데이트되고 페이지의 올바른 언어 버전이 표시됩니다.

비디오 속성을 편집하려면 섹션에서편집버튼을 클릭하거나 섹션을 두 번 클릭합니다. 지역화된 값을 입력한 다음저장을 클릭합니다 .

다음 비디오 속성 섹션은 다국어 메타 데이터를 지원합니다.

  • 비디오 정보
  • 오디오 트랙
  • 이미지
  • 사용자 정의 필드
  • 텍스트 트랙

필수 입력 사항

변경 사항을 저장하려면 기본 언어의 모든 필수 필드에 값이 있어야합니다. 추가 언어에 대한 필수 필드가 누락 된 경우 메시지가 표시되지만 변경 사항을 저장할 수 있습니다.
