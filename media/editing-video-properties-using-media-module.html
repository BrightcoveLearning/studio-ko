비디오 속성을보고 편집하려면 미디어 모듈을 열고 비디오 제목 링크를 클릭합니다. 비디오 속성 페이지가 열립니다.

참고: 여러 비디오의 비디오 속성을 동시에 편집하려면빠른 편집기능을 사용합니다.

비디오 속성 페이지 상단의 버튼은 다음과 같은 기능을 제공합니다.

  • 새 포스터 및 썸네일 이미지 캡처
  • 동영상을 폴더로이동하거나 재생 목록에추가
  • 비디오 비활성화 / 활성화
  • 비디오 게시

비디오가 활성화되면비활성화버튼을 사용하여 비디오를 비활성화할 수 있습니다. 비디오가 비활성화된 경우, 활성화버튼을 사용하여 비디오를 활성화할 수 있습니다. 비활성화 된 비디오는 Brightcove Player에서 재생할 수 없습니다.

페이지 상단의 미리보기 플레이어를 사용하여 비디오를 볼 수 있습니다.

비디오 속성은 다음 섹션으로 구성됩니다.

  • 비디오 정보
  • 분류
  • 비디오 파일
  • 미디어 공유
  • 오디오 트랙
  • 이미지
  • 비디오 조회수
  • 사용자 정의 필드
  • 유효성
  • 재생 제한
  • 텍스트 트랙

비디오 속성을 편집하려면 섹션에서편집버튼을 클릭하거나 섹션을 두 번 클릭합니다. 일부 섹션에서는 속성을 직접 편집할 수 있습니다 (예: 범주화).

필수 입력 사항

Video Cloud 계정은 필수 메타 데이터 필드로 구성 될 수 있습니다. 비디오가 비활성 상태 인 경우 필요한 모든 메타 데이터를 제공하지 않고도 메타 데이터를 자유롭게 편집하고 저장할 수 있습니다. 또한 모든 필수 메타 데이터를 입력 할 때까지 비디오를 활성화 할 수 없습니다. 비디오가 이미 활성화 된 경우 해당 섹션에 필요한 모든 메타 데이터를 제공하거나 비디오를 비활성화하지 않는 한 편집중인 섹션의 편집 내용을 저장할 수 없습니다. 필수 필드 작업에 대한 자세한 내용은 비디오의 필수 필드설정을 참조하십시오 .

비디오 정보

비디오 정보 섹션은 기본 비디오 속성을보고 편집하는 데 사용할 수 있습니다.

등록 정보를 편집하려면편집버튼을 클릭하거나 섹션을 두 번 클릭합니다. 다음 속성을 편집 할 수 있습니다.

  • 이름 -255 자 이하.
  • 신분증 -시스템 할당 비디오 ID (모든 Video Cloud 계정에서 고유함).
  • 간단한 설명 -250 자 이하.
  • 태그 -쉼표로 구분 된 하나 이상의 태그. 태그를 사용하여 스마트 재생 목록에 비디오를 추가하거나 CMS API를 사용하여 비디오를 찾을 수 있습니다.
    • 태그는 최대 128 자까지 가능합니다. 128 자 이상의 태그가있는 경우 변경 사항을 저장하려고하면 오류가 표시됩니다.
    • 태그는 쉼표를 포함 할 수 없습니다. 예를 들어 "지붕, 다락방 + 앞뜰"태그는 지원되지 않습니다.
    • 동영상에는 총 1200 개의 태그를 포함 할 수 없습니다.
    • 대소 문자 만 다른 태그 (소문자, 대문자)는 동일한 태그로 처리되며 대소 문자는 원래 태그와 자동으로 일치합니다.
    • 대소 문자 구분 문제는 JavaScript 메서드를 사용하여 문자열을 수정하는 등 Video Cloud 외부에서 처리해야합니다.
    • 태그를 입력한 후반점키 또는시작하다 . 추가 된 태그는 컬러 상자에 표시됩니다.
  • 참조 ID -게시자가 동영상 또는 재생 목록에 할당 한 고유 ID입니다. 150 자 이하로 제한됩니다. 참조 ID는 Video Cloud에서 생성 된 고유 ID 인 비디오 ID 또는 재생 목록 ID와 다릅니다. 참조 ID를 사용하면 Video Cloud와 게시자의 콘텐츠 관리 시스템과 같은 다른 서비스를 조정할 수 있습니다. 두 개의 동영상이 동일한 참조 ID를 가질 수 없으므로 참조 ID를 일괄 편집 할 수 없습니다. 참조 ID는 항상 고유해야합니다. 동영상을 삭제하면 이전에 할당 된 참조 ID를 재사용 할 수 있습니다.
  • 비디오 프로젝션 -비디오 녹화 방법을 나타냅니다.
    • 표준 - 기본. 정기적 인 비디오 녹화.
    • 360도 -동영상이 360도 동영상임을 표시하려면 선택하세요. 360 배지는 미디어 모듈의 비디오 섬네일에 나타납니다. 360도 비디오 작업에 대한 자세한 내용은 360° 비디오작업을 참조하십시오 .
  • 오프라인 재생 활성화 -체크하면 장치 SDK를 사용하여 만든 응용 프로그램을 사용할 때 비디오를 로컬에 저장하고 오프라인으로 볼 수 있습니다. 이 옵션은 계정이 오프라인 재생을 위해 활성화 된 경우에만 나타납니다.
  • 관련 링크 URL - http://또는 https: //로 시작하는 표준 웹 URL은 255자로 제한됩니다.
  • 관련 링크 텍스트 -관련 링크 텍스트는 255 자로 제한됩니다.
  • 장황한 설명 -5000 자 이하 포함 된 탭 및 HTML은 지원되지 않습니다.
  • 광고하는
    • 비어 있는 -동영상 재생시 광고가 표시되지 않습니다.
    • 광고 지원 -동영상 재생시 광고가 표시 될 수 있습니다.
      참고: 계정에 광고가 활성화되어 있지 않고 광고 속성을 광고 지원됨으로 설정하면 변경 사항을 저장할 때 오류가 표시됩니다.
  • 광고 키 -광고 서버 URL의 일부로 전달됩니다. 예를 들어 통과하려면section=blog그리고anotherKey=value1 , 당신은 입력합니다section=blog&anotherKey=value1 . 참조 DoubleClick for Publishers 도움말키-값 쌍 추가에 대한 정보는 광고 키를 사용하는 경우 광고 매크로를 광고 서버 URL ( 플레이어 광고 속성의 일부로 구성 ) 에 추가해야 합니다. 광고 매크로는 IMA3 플러그인이 적절한 값으로 대체 할 서버 URL의 변수를 지원합니다. 자세한 내용은
  • 나누는 -동영상이 다른 계정과 공유되었는지 여부를 나타냅니다. 자세한 내용은 미디어 모듈을 사용한미디어 공유를 참조하십시오 .

분류

분류 섹션에는 다음이 표시됩니다. 수동 재생 목록비디오가 할당되고폴더비디오가 있습니다. 속성을 편집하려면이 섹션에서 직접 변경하십시오. 변경 사항을 저장하려면 Enter를 누르십시오.

  • 수동 재생 목록 -이 동영상이 속한 재생 목록. X를 클릭하여 재생 목록에서 비디오를 제거합니다. 자세한 내용은 재생 목록만들기 및 관리를 참조하십시오 .
  • 폴더 -동영상이 할당 된 폴더입니다. 보다폴더를 사용하여 비디오 정리자세한 내용은.

Enter를 누르면 변경 사항이 저장됩니다.

비디오 파일

비디오 파일 섹션에는 소스 파일 이름, 사용 된 트랜스 코딩 기술 및 생성 된 비디오 변환에 대한 세부 정보가 표시됩니다. 또한 다음 기능을 제공합니다.

  • 비디오 소스 파일 교체
  • 비디오 재 트랜스 코딩

이 예에서는동적 전송트랜스코딩이 사용되었습니다 (비디오가 업로드될 때 동적 전송 또는 컨텍스트 인식 인코딩 인제스트 프로파일이 사용되었습니다).

참고: 계정이 DRM에 대해 활성화 된 경우 DRM 토글 스위치가 섹션에 나타날 수도 있습니다. DRM에 대한 자세한 내용은 DRM을 사용하여 콘텐츠수집을 참조하십시오 .

미디어 공유

동영상이 제휴 계정에 공유 된 경우 미디어 공유 섹션이 표시되고 동영상이 공유 된 제휴사가 나열됩니다. 미디어 공유에 대한 자세한 내용은미디어 모듈을 사용한 미디어 공유를 참조하십시오 .

오디오 트랙

비디오가 Dynamic Delivery 또는 Context Aware Encoding 수집 프로파일을 사용하여 업로드되고 트랜스 코딩 된 경우 오디오 트랙 섹션이 표시되고 사용 가능한 오디오 변환에 대한 정보가 포함됩니다. 비디오에 오디오 트랙을 추가하는 방법에 대한 자세한 내용은 미디어 모듈을 사용하여 비디오에 오디오 트랙추가를 참조하십시오 .

참고: 비디오가 레거시 인제스트 프로파일을 사용하여 트랜스코딩된 경우에는 오디오 트랙 섹션이 표시되지 않습니다.

이미지

이미지 섹션에는 비디오의 포스터 및 썸네일 이미지가 표시됩니다. 포스터 및 썸네일 이미지는 중간 지점에서 소스 비디오에서 스냅 샷을 찍어 자동으로 생성됩니다. 캡처된 기본 이미지가 마음에 들지 않으면 다음을 수행할 수 있습니다. 새로운 이미지 캡처또는다른 이미지 업로드포스터 및 썸네일로 사용합니다. 포스터/썸네일 이미지 위로 마우스를 가져간 다음 보기 아이콘() 새 브라우저 탭에서 이미지를 엽니다.

비디오 조회수

동영상 조회수 섹션에는 지난 30 일 동안의 동영상 조회수가 표시됩니다. 자세한 내용을 보려면 링크를 클릭하여 성과 또는 참여 보고서를여십시오.

사용자 정의 필드

사용자 정의 필드 섹션은 비디오의 사용자 정의 필드를보고 편집하는 데 사용할 수 있습니다. 사용자 정의 필드 값을 편집하려면편집버튼을 클릭하거나 섹션을 두 번 클릭합니다. 필드 이름 옆에 별표 (*)는 필드가 필수임을 나타냅니다.

참고: 계정에 사용자 지정 필드를 추가하는 방법에 대한 자세한 내용은 사용자 지정 메타데이터 필드만들기를 참조하십시오 .

유효성

가용성 섹션에는 비디오 가용성 설정이 표시됩니다. 가용성 설정 구성에 대한 자세한 내용은 비디오 가용성구성을 참조하십시오 .

재생 제한

재생 제한 섹션에는 재생 제한이 표시됩니다. 재생 제한 구성에 대한 자세한 내용은 재생 권한관리를 참조하십시오 .

텍스트 트랙

텍스트 트랙 섹션은 비디오에 텍스트 트랙을 추가하는 데 사용할 수 있습니다. 텍스트 트랙 구성에 대한 자세한 내용은 미디어 모듈을 사용하여 비디오에 캡션추가를 참조하십시오 .
