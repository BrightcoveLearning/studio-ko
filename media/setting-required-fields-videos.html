많은 게시자가 동영상 메타 데이터 (이름, 태그, 참조 ID, 맞춤 입력란 등)를 콘텐츠를 추적하고 수익을 창출하기위한 필수 도구로 사용합니다. 비디오 메타 데이터가 효과적이려면 의존하는 각 메타 데이터 필드가 실제로 존재하는지 확인해야합니다. 메타 데이터 보증 기능을 사용하면 비즈니스 목적에 중요한 비디오 메타 데이터 필드를 지정한 다음 사용자가 비디오를 활성화하기 전에 새 비디오에 필요한 메타 데이터 필드를 완료하도록 요구할 수 있습니다.

필요에 따라 메타 데이터 필드 지정

필수 비디오 메타 데이터 필드는 Video Cloud 계정 설정의 일부로 구성됩니다. 표준 및 사용자 지정 메타 데이터 필드 모두 필수로 표시 할 수 있습니다. 필수 필드를 구성하려면 다음과 같이 하십시오.

  1. Studio 헤더에서관리링크를 클릭합니다.
  2. 비디오 필드링크를 클릭합니다.
  3. 필요에 따라 설정하려는 필드 이름을 클릭합니다.
  4. 을 체크 해봐이 필드를 필수로 설정체크박스.
  5. 필드저장을 클릭합니다 .

언제든지 선택을 취소할 수 있습니다. 이 필드를 필수로 설정해당 필드가 더 이상 필요하지 않게 하려면 해당 필드에 대한 확인란을 선택합니다. 그 시점부터 Video Cloud는 해당 필드가 채워 졌는지 더 이상 확인하지 않습니다.

필수 필드가 될 수있는 메타 데이터 필드는 무엇입니까?

비디오 이름 필드는 항상 필수입니다. 모든 사용자 지정 메타 데이터 필드는 필수 필드가 될 수 있습니다. 또한 다음과 같은 표준 메타 데이터 필드가 필요할 수 있습니다.

  • 광고하는
  • 지리적 필터링 켜기
  • 간단한 설명
  • 장황한 설명
  • 참조 ID
  • 관련 링크 텍스트
  • 관련 링크 URL
  • 예약 시작 날짜
  • 예약 종료 날짜
  • 태그

필수 메타 데이터가없는 동영상

미디어 라이브러리의 비디오에 필수 메타 데이터가 모두없는 경우 다음을 수행하십시오.

  • 이미 활성화되어 있으면 계속 활성화되고 재생할 수 있습니다. 즉, 계정에 필요한 메타 데이터 필드를 지정할 때 필요한 메타 데이터를 업데이트 할 때까지 기존 비디오 라이브러리를 갑자기 재생할 수 없게되지 않습니다.
  • 이미 활성화되어 있고 미디어 모듈에서 편집하는 경우 편집중인 섹션에 필요한 모든 메타 데이터를 제공하지 않고 편집 내용을 저장할 수 없습니다.
  • 비활성 상태 인 경우 필요한 모든 메타 데이터를 제공하지 않고 활성화 할 수 없습니다.

필수 메타 데이터없이 동영상 업로드

기본적으로 업로드 모듈을 사용하여 비디오를 업로드할 때 비디오의 상태는 다음으로 설정됩니다. 활동적인필수 필드가 누락되지 않는 한. 필수 필드가 누락 된 경우 필수 필드가 빨간색으로 강조 표시된 메시지가 표시됩니다. 필수 입력란이 누락된 동영상은 계속 업로드할 수 있지만 그대로 유지됩니다. 비활성모든 필수 필드에 값이 있을 때까지

비디오 편집

비디오가 비활성 상태 인 경우 필요한 모든 메타 데이터를 제공하지 않고도 미디어 모듈에서 해당 메타 데이터를 자유롭게 편집하고 저장할 수 있습니다. 또한 필요한 모든 메타 데이터를 입력 할 때까지 비디오를 활성화 할 수 없습니다.

비디오가 이미 활성화 된 경우 해당 섹션에 필요한 모든 메타 데이터를 제공하거나 비디오를 비활성화하지 않는 한 편집중인 섹션의 편집 내용을 저장할 수 없습니다.

비디오 활성화

미디어 모듈에서 비디오를 활성화하려면 필요한 모든 메타 데이터가 있어야합니다. 필수 메타 데이터가 누락 된 동영상을 활성화하려고하면 오류가 표시됩니다.

하단의 상태 표시 줄은 동영상을 활성화 할 수 있는지 여부를보고합니다. 활성화 버튼은 비디오에 필요한 모든 메타 데이터를 제공 할 때까지 비활성화 된 상태로 유지됩니다.

워크 플로 팁

모든 비활성 동영상을 찾고 싶으십니까? 미디어 모듈에서 모든 동영상을상태별로 정렬할 수있습니다. 상태열의 헤더를 클릭하기만 하면 됩니다. 그런 다음 각 비활성 비디오를 편집하여 누락 된 필수 메타 데이터를 확인할 수 있습니다.

한계

필수 필드는 미디어 모듈에서만 적용됩니다. 필수 메타 데이터 필드가 모두 완료되지 않으면 미디어 모듈에서 비활성 비디오를 활성화 할 수 없습니다. 그러나 필수 메타 데이터 필드가없는 비디오는 다음과 같은 방법으로 계속 활성화 될 수 있습니다.

  • 동영상은 계정에서 생성되었으며 계정에 대한 필수 필드를 지정하기 전에 활성화되었습니다.
  • 동영상이 Dynamic Ingest API를 사용하여 업로드되었습니다.

또한 Video Cloud는 유형의 유효성을 검사하지만 필수 메타 데이터의 값은 유효성을 검사하지 않습니다. 제공된 값에 계정 설정 (예: 텍스트, URL 또는 날짜) 의 필드에 대해 올바른 유형이 나열되어 있으면 사용자가 선택한 값을 입력할 수 있습니다.
