Video Cloud 계정에서 비디오를 다운로드 할 수있는 방법에는 여러 가지가 있습니다. 비디오는 API를 사용하여 미디어 또는 소셜 모듈을 사용하여 다운로드 할 수 있으며 시청자가 콘텐츠를 다운로드 할 수 있도록 플레이어에 다운로드 링크를 추가 할 수 있습니다.

미디어 모듈을 사용하여 비디오 다운로드

미디어 모듈을 사용하여 비디오 변환을 다운로드 할 수 있습니다. 미디어 모듈의 비디오 이름 옆에 있는 구름 아이콘이 회색(), 비디오는 레거시 수집 프로필을 사용하여 수집되었으며 MP4 및 HLS 변환 모두 미디어 모듈에서 다운로드할 수 있습니다. 녹색 구름 아이콘()는 비디오가 Dynamic Delivery를 사용하여 수집되었음을 나타내며 MP4 변환을 생성하는 수집 프로필을 선택한 경우 MP4 변환을 다운로드할 수 있습니다.

노트:
  • 동영상 속성 페이지의 동영상 파일 섹션에는 사용 된 수집 프로필 유형도 표시됩니다.
  • 1080p 변환을 다운로드할 수 있도록 하려면 사용자 정의 수집 프로필을만들고 다음 변환 중 하나 이상을 포함해야 합니다.
    • 프로그레시브 3500
    • 프로그레시브 4000

미디어 모듈을 사용하여 비디오를 다운로드하려면 다음 단계를 따르십시오.

  1. 미디어모듈을 엽니다.
  2. 다운로드하려는 비디오 링크를 클릭하십시오.
  3. 비디오 파일섹션을 찾습니다.
  4. 사용 된 코드 변환 유형을 확인하십시오.
    • 동적 배달 (녹색 구름 아이콘, 아래 예) - 다운로드할 변환에 대한차원열의 링크를 마우스 오른쪽 버튼으로 클릭합니다. 클릭 가능한 다운로드 링크는 MP4에서만 표시됩니다.변환이 작성되었습니다.
    • 레거시 (회색 구름 아이콘) - 다운로드할 변환에 대한형식열의 링크를 마우스 오른쪽 버튼으로 클릭합니다.
참고: Dynamic Delivery에서 생성 된 변환의 TTL (Time to Live)은 6 시간입니다. 생성된 링크는 6시간 동안 유효합니다.

변환 및 이미지에 대한 URL이 고정되어 있지 않음을 이해하는 것이 중요합니다. Brightcove는 수시로 미디어 자산의 저장소를 재구성하며 이러한 상황이 발생하면 특정 자산의 URL이 변경됩니다. 이러한 자산에 대해 하드 코딩 된 URL을 사용하는 경우 링크가 어느 시점에서 끊어 질 수 있습니다. 미디어 링크가 끊어지지 않도록 하는 가장 좋은 방법은 런타임에 CMS API를 사용하여 Video Cloud에서 미디어를 검색하는 것입니다.

Social 모듈을 사용하여 비디오 다운로드

Social 모듈은 비디오 URL을 복사하거나 비디오 소스 파일을 다운로드하는 기능을 제공합니다. 다운로드 할 때 최고 품질의 MP4 변환이 다운로드됩니다.

Social 모듈을 사용하여 비디오를 다운로드하려면 다음 단계를 따르십시오.

  1. Brightcove Social을 엽니 다.
  2. 딸깍 하는 소리비디오왼쪽 탐색에서 Video Cloud 계정의 비디오 목록이 표시됩니다.
  3. 다운로드할 비디오를 클릭한 다음다운로드를 클릭합니다 .
    참고: MP4 변환이 없으면다운로드링크가 비활성화됩니다.
    참고: DRM이 활성화 된 비디오는 다운로드 할 수 없습니다 (비디오 섬네일에 잠금이 나타남). DRM 사용 비디오를 선택하면다운로드버튼이 비활성화됩니다.
  4. 비디오 URL을 복사하거나 링크를 클릭하여 브라우저에서 비디오를 엽니 다. 비디오를 마우스 오른쪽 버튼으로 클릭하여 소스 파일을 로컬에 저장합니다.
  5. 완료를 클릭합니다 .

CMS API를 사용하여 비디오 속성에 액세스

NS CMS API Video Cloud 계정의 모든 비디오에 대해 비디오 URL을 포함한 메타데이터를 검색하는 데 사용할 수 있습니다. CMS API를 사용하려면 개발 기술이 필요합니다. CMS API를 사용하여 비디오 속성에 액세스하는 방법에 대한 자세한 내용은 CMS API 사용을 참조하십시오. 비디오 데이터개발자 문서 검색

Brightcove Learning Services도 샘플을 만들었습니다. CMS API다운로드 링크를 얻는 데 사용할 수 있는 앱: CMS API 샘플: 다운로드 링크 .

비디오 파일에 대한 다운로드 링크 만들기

개발 기술이있는 경우 플레이어 컨트롤 막대에 다운로드 링크를 배치하여 시청자에게 콘텐츠 다운로드 옵션을 제공 할 수 있습니다.

플레이어 오버레이 또는 플레이어 아래에 다운로드 링크를 표시 할 수도 있습니다. 샘플 코드를 포함하여 이 작업을 수행하는 방법에 대한 자세한 내용은브라이트코브 플레이어 샘플을 참조하십시오. 비디오 플러그인 다운로드 .
