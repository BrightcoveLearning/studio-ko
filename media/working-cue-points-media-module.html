큐 포인트는 비디오 재생 중 정확한 시점에있는 마커입니다. 큐 포인트를 사용하여 미드 롤 광고 (즉, 광고 시간과 같이 비디오 콘텐츠를 재생하는 동안 발생하는 광고)를 트리거하거나 긴 형식의 비디오에서 장 또는 장면을 분리 할 수 있습니다.

큐 포인트에는 두 가지 유형이 있지만 둘 다 큐 포인트 객체의 type속성에 문자열 값을 제공하기만 하면 됩니다. 적절한 값은 큐 포인트 정보를 처리하는 동안 사용자 정의 JavaScript에서 사용되는 경우에만 유용합니다.  

  • Ad - 유형속성에 AD의 문자열 값을할당합니다.
  • 암호 - 문자열 값을 할당합니다. 암호 ~로유형재산

큐 포인트는 미디어 모듈을 사용하거나 카탈로그 API를 사용하여 비디오에 추가 할 수 있습니다. 카탈로그 API를 사용하여 큐 포인트를 추가하는 방법에 대한 자세한 내용은 프로그래밍 방식으로 큐 포인트구현 개발자 문서를 참조하십시오 .

참고: 큐 포인트는 라이브 원격 에셋 비디오에 추가할 수 없습니다.

큐 포인트 추가

비디오에 큐 포인트를 추가하려면 미디어 모듈을 열고 다음 단계를 따르십시오.

  1. 비디오 제목 링크를 클릭하여 비디오 속성 페이지를 엽니 다.
  2. 미리보기 플레이어를 사용하여 비디오를 재생하십시오. 큐 포인트를 추가하려는 지점에서 비디오를 일시 중지합니다.
  3. 큐 포인트추가를 클릭합니다 .
  4. 필요한 경우 큐 포인트의시간 코드 (hh:mm:ss)값을 조정합니다. 초필드는 소수점 이하 밀리초를 지원합니다.
  5. 큐 포인트유형을 선택한 다음 적절한 메타데이터를 추가합니다.
    • 기원 후 -광고 요청이 생성됩니다. 광고 큐 포인트를 사용할 때는 광고용으로 플레이어를구성해야합니다.
    • 암호 -큐 포인트는 프로그래밍 방식으로 처리됩니다. 자세한 내용은 프로그래밍 방식으로 큐 포인트구현개발자 문서를 참조하십시오.
  6. 큐 포인트의이름을입력합니다. name 값은프로그래밍 방식으로 액세스할 수있습니다.
  7. 메타데이터 (코드 큐 포인트의 경우) 또는키/값 쌍 (광고 큐 포인트의 경우) 을 입력합니다.
    • 메타 데이터 -큐 포인트와 함께 저장된 선택적 메타 데이터입니다. 프로그래밍 방식으로 메타데이터에 액세스할 수있습니다. 코드 큐 포인트를 사용하고 메타 데이터를 전달하려는 경우의 한 가지 예는 비디오가 재생 될 때 다른 클릭 유도 문안을 표시하려는 경우입니다. 브라이트코브 플레이어 샘플 보기: 큐 포인트는 예를들어 CTA를 표시합니다.
    • 키 / 값 쌍 -광고 요청에 전달할 수있는 키 / 값 쌍입니다. 키-값 쌍은 두 개의 연결된 데이터 항목 집합으로, 일부 데이터 항목에 대한 고유 식별자 인 키와 식별 된 데이터 또는 해당 데이터 위치에 대한 포인터 인 값입니다. 키 / 값 쌍을 사용하여보다 타겟팅 된 광고를 제공 할 수 있습니다. 여기에 입력한 내용이{mediainfo.ad_keys}매크로가 삽입된 광고 태그에 복사됩니다. 자세한 내용은 IMA3 플러그인을 사용한광고를 참조하십시오 .

큐 포인트가 비디오에 추가 된 후 런타임에 큐 포인트가 HTML5 WebVTT 큐 요소로 변환됩니다. 이 변환이 발생하면 유형 및 큐 포인트 시간과 같은 일부 큐 포인트 데이터가 텍스트 트랙으로 변환됩니다. 큐 포인트 시간 값은 WebVTT 큐startTime및endTime속성에 매핑됩니다. 보다이 문서 VTTCue 인터페이스에 대한 자세한 내용은

큐 포인트 편집

비디오에서 큐 포인트를 편집하려면 미디어 모듈을 열고 다음 단계를 따르십시오.

  1. 비디오 제목 링크를 클릭하여 비디오 속성 페이지를 엽니 다.
  2. 타임 라인에서 편집 할 큐 포인트를 클릭합니다.
    참고: 편집 모드에서이전및다음버튼을 사용하여 다른 큐 포인트로 이동할 수 있습니다.
  3. 큐 포인트 속성을 편집한 다음저장을 클릭합니다 .
참고: 동일한 시간 코드를 사용하여 여러 큐 포인트를 가질 수는 없습니다.

큐 포인트 삭제

큐 포인트를 삭제하려면 미디어 모듈을 열고 다음 단계를 따르십시오.

  1. 비디오 제목 링크를 클릭하여 비디오 속성 페이지를 엽니 다.
  2. 타임 라인에서 삭제할 큐 포인트를 클릭합니다.
  3. 삭제 아이콘() 큐 포인트 속성 옆에 있습니다.
  4. 삭제를 확인하십시오.
