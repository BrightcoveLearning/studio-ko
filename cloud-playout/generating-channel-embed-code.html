Cloud Playout 채널 용 소스 코드 생성

Cloud Playout 채널의 소스 코드를 생성하려면 Live 모듈을 사용합니다. 라이브 모듈 사용에 대한 자세한 내용은 라이브 모듈을 사용하여 라이브 이벤트만들기 및 관리를 참조하십시오 .

두 가지 유형의 소스 코드를 사용할 수 있습니다.

  • 표준 -iframe 소스 코드. 대부분의 경우에 권장됩니다. 이 유형은 플레이어를 <iframe> 태그 안에 배치합니다.
  • 많은 -인 페이지 소스 코드. 플레이어가 iframe이 아닌 HTML 페이지에 직접 존재하도록 허용합니다. 이를 통해 개발자는 플레이어 및 관련 속성 및 이벤트에 쉽게 액세스 할 수 있습니다.

게시 된 플레이어에 대한 URL을 가져올 수도 있습니다. 이것은 브라우저에서 이벤트를 보는 데 유용합니다.

라이브 이벤트에 대한 소스 코드를 생성하려면 다음 단계를 따르세요.

  1. 열기라이브기준 치수.
  2. 채널이 스트리밍 될 라이브 이벤트를 찾습니다. 일반적으로채널이 생성될 때 스트리밍할 특정 이벤트를 선택하지않은 경우 이벤트는대기탭에 위치하며 채널 이름이 지정됩니다.
  3. 이벤트 이름을 클릭하여 Control Room페이지를 엽니다.
  4. 딸깍 하는 소리게시 및 포함> 플레이어 구성왼쪽 탐색에서
    참고: 채널이 서버 측 광고 (SSAI) 를 사용하도록 구성된 경우 브라이트코브 플레이어 버전 6.18.3 이상을사용해야 합니다. 버전 6.18.3 이상의 플레이어만 드롭다운 목록에 나타납니다.
  5. 플레이어 URL을클릭하여 브라우저에서 이벤트를 미리 보거나 사용할 적절한 임베드 코드를 복사합니다.

광고 삽입옵션이켜기로 설정된 프로그램을 만든 경우 게시 옵션에 광고 구성선택드롭다운이 포함됩니다.사용할 광고 구성을 선택할 수 있습니다.
