Brightcove Social은 소셜 미디어 계정에 게시 된 비디오의 성능을 모니터링하는 데 사용할 수있는 여러 보고서를 제공합니다.

  • 계기반 -소셜 미디어 계정의 비디오 성능에 대한 높은 수준의 개요를 제공합니다. 자세한 내용은 소셜 분석검토를 참조하십시오 .
  • 사회적 견해 -소셜 미디어 계정에 공유 된 동영상에 대한 소셜보기의 개요를 제공합니다. 이 주제에서 다룹니다.
  • 사회적 참여 -소셜 미디어 계정에 공유 된 동영상에 대한 소셜 참여 (좋아요, 공유, 리트 윗 등)에 대한 개요를 제공합니다. 자세한 내용은 소셜 참여 분석검토를 참조하십시오 .

먼저보고서의 시간 범위를선택합니다. 소셜 뷰 분석 보고서를 보려면 페이지 상단의소셜 뷰탭을 클릭합니다.

Social View Analytics 보고서는 다음 섹션으로 구성됩니다.

  • 사회적 견해
  • 소셜 뷰 세부 정보

사회적 견해

소셜 조회수 그래프는 선택한 기간 동안 소셜 미디어 계정에 게시 된 동영상의 동영상 조회수를 표시합니다. 업데이트 날짜는 Video Cloud에 소셜 분석 데이터가있는 최신 날짜입니다. 일반적으로 소셜 플랫폼이 API에서보기 및 참여 데이터를 업데이트하기까지 24-48 시간 이상 지연됩니다.

참고: Twitter Insights API는 미디어 조회수를 보고하지 않으므로 소셜 분석의 일부로 데이터를 볼 수 없습니다.

그래프 오른쪽의 범례를 사용하여 표시 할 세부 정보 수준을 선택할 수 있습니다.

  • 내 사이트 및 앱 -Video Cloud 플레이어 및 앱의 비디오 조회수 표시
  • 페이스 북 -Facebook에 비디오 조회수 표시
  • 유튜브 -YouTube에서 동영상 조회수 표시
  • 합계 -모든 플랫폼에서 총 비디오 조회수 표시

그래프의 선 위로 마우스를 가져 가면 해당 날짜에 대한 데이터가 표시됩니다.

소셜 뷰 세부 정보

소셜보기 세부 정보 섹션을 사용하여 다음과 같은 방법으로 비디오보기 세부 정보를 볼 수 있습니다.

  • 보기 (기본값)
  • 평균 시청 시간
  • 평균 시청률

선택이 끝나면 보고서에 각 비디오에 대한 자세한 데이터가 표시됩니다.

참고: Twitter Insights API는 미디어 조회수를 보고하지 않으므로 소셜 분석의 일부로 데이터를 볼 수 없습니다.

분석 보고서에 표시된 YouTube 조회수는 YouTube 분석 개요보고서에서 수집됩니다. YouTube는이 데이터가 실시간 데이터임을 보장하지 않습니다. 따라서이 보고서에 조회수와 참여가 표시되기까지 며칠이 지연 될 수 있습니다. 데이터가 YouTube 분석에 있으면 Brightcove 소셜 분석을 위해 야간 데이터 수집가가 데이터를 수집합니다.

최신 실시간 시청 데이터가 필요한 경우 YouTube는 최근 게시된 상위 25개 동영상에 대해 지난 48시간 동안의 예상 데이터를 보고 있는실시간 보고서를제공합니다. YouTube 채널의 실시간 보고서는여기에서 확인할 수있습니다. Brightcove Social Analytics는 예상 조회수 (실제 조회수와 반대)를 제공하고 최근 동영상에만 적용되므로이 실시간 데이터를 수집하지 않습니다.

보고서 하단에는 표시 할 행 수를 선택하고 다른 페이지로 이동할 수있는 컨트롤이 있습니다. 비디오 타이틀 링크를 클릭하면 선택한 비디오에 대한 소셜 조회수 분석이 표시됩니다.

일반적인 질문

1. 소셜 모듈을 사용하여 동영상을 YouTube에 게시 한 다음 해당 동영상이 다른 사이트의 YouTube 플레이어에 삽입되면 재생 / 참여 데이터가 수집되어 소셜 모듈 분석에 전달됩니까?

소셜 분석은 크리에이터 스튜디오 및 / 또는 YouTube의 분석 페이지에 표시되는 YouTube에서보고 한 모든보기 데이터를 수집합니다. YouTube 내장 플레이어 조회수가 분석에 포함되면 소셜 모듈이이를 캡처하여보고합니다. 삽입 코드 YouTube의공유 > 삽입기능을 제공된 대로 정확하게 사용하면 포함된 플레이어의 조회수가 YouTube에서 내 분석에 반영됩니다. 그러나 소스 코드를 편집하면 YouTube가 조회수 데이터를 수집하지 않을 수 있으며 특히 내장 플레이어가 동영상을 자동 재생하도록 설정하면 YouTube가 해당 플레이어의 조회수를 계산하지 않는 것 같습니다.

데이터 내보내기

보고서 데이터는 쉼표로 구분 된 값과 Excel 형식으로 다운로드 할 수 있습니다. 내보내기 기능은 에서 직접 데이터를 저장합니다. 애널리틱스 API전화. 내 보낸 데이터는 UI 데이터 방식대로 조작되지 않으며 현재 UI에 표시되는 필드의 하위 집합뿐 아니라 보고서의 모든 기본 데이터를 포함합니다. API의 모든 데이터를 원래 형식으로 가져 오면 UI에서 제공하는 것과 다른 방식으로 데이터를 조작 할 수 있습니다.
