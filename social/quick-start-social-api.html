소개

Social API를 사용하면 소셜 미디어 플랫폼에 비디오를 공유 한 상태와 기록을 얻을 수 있습니다. 이 빠른 시작을 통해 자신의 계정 또는 Brightcove Learning Services 샘플 계정을 사용하여 사용 가능한 요청과 응답을 볼 수 있습니다.

샘플 앱에 대한 참고 사항

  • 이것은 Social API를 중심으로 앱을 빌드하기위한 모델을 제공하기위한 빠른 시작입니다. 기본 요청을 테스트 할 수 있지만 이러한 요청이 사용할 수있는 매개 변수는 사용하지 않습니다. 이를 확인하려면 API 레퍼런스를 살펴보십시오 .
  • 앱은 또한 CMS API비디오 소셜 공유 상태에 대한 초기 요청에서 반환된 비디오 이름을 가져옵니다. 이는 Social API가 이름이 아닌 동영상 ID 만 반환하기 때문에 다른 요청에 대한 동영상 선택기를 채우기위한 것입니다.
이 샘플 및 관련 코드는 프로덕션 개발을 위한 가이드로 제공됩니다. 이 코드는 프로덕션 용도로 인증되지 않았으며 공식적으로 지원되지 않습니다.

자격 증명 받기

Social API를 사용하려면 적절한 자격 증명이 필요합니다.

가장 쉬운 방법은 Studio Admin/API 인증섹션을 사용하여 새 애플리케이션등록을 클릭한 다음 자격 증명에 대한 소셜 권한을 선택하는 것입니다.

프록시 코드

이 페이지에서 샘플 앱을 자체 버전으로 빌드하려면 자체 프록시를 만들고 호스팅해야합니다. (Brightcove Learning Services에서 사용하는 프록시는 Brightcove 도메인의 요청 만 수락합니다.) 샘플 프록시는 우리가 사용하는 것과 매우 유사하지만 브라이트코브 도메인이 아닌 도메인의 요청을 차단하는 검사가 없으면이 GitHub 저장소에서 찾을 수있습니다. 또한 여기에서 사용에 대한 기본 지침과 프록시 주변에서 앱을 빌드하는 더 자세한 가이드를 찾을 수 있습니다. REST API 사용 .
