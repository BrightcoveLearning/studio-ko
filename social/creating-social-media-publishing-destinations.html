게시 대상은 Video Cloud 비디오를 게시하려는 소셜 미디어 계정 / 페이지입니다. Brightcove Social을 사용하여 소셜 미디어 계정에 비디오를 게시하려면 먼저 하나 이상의 대상을 만들어야합니다. YouTube, Facebook 및 Twitter에 비디오를 게시하기 위해 대상을 만들 수 있습니다.

YouTube 대상 만들기

YouTube 채널에 비디오를 게시하려는 경우 YouTube 대상을 만듭니다. YouTube 목적지를 만들기 전에 YouTube 계정을확인해야 할 수있습니다. 계정을 확인하면 15 분 이상의 동영상을 업로드 할 수 있습니다.

YouTube 대상을 만들려면 다음 단계를 따르세요.

  1. Brightcove Social을 엽니 다.
  2. 딸깍 하는 소리목적지왼쪽 탐색에서
  3. 대상추가를 클릭합니다 .
  4. YouTube로로그인을 클릭합니다 .
  5. Google 계정에 로그인하라는 메시지가 표시됩니다 (아직 로그인하지 않은 경우). 계정이 여러 개인 경우 YouTube 채널과 연결된 계정에 로그인해야합니다.
  6. YouTube 계정 관리 권한을 요청하는 대화 상자가 열립니다. 대화 상자의 오른쪽 상단을 확인하여 올바른 Google 계정을 사용하고 있는지 확인하십시오. 그렇지 않은 경우 드롭 다운을 사용하여 동영상을 배포 할 Google 계정을 선택합니다. 허용을 클릭합니다 .
  7. 목적지에 고유하고 설명이 포함 된 이름을 지정하십시오. 추가를 클릭합니다 .
  8. 기본 정보와 목적지 기본값이 표시됩니다. 기본 정보섹션에서편집버튼을 클릭하여 대상 이름을 편집합니다. 기본값 구성에 대한 자세한 내용은 YouTube 기본값설정섹션을 참조하십시오.
  9. 딸깍 하는 소리목적지을 클릭하고 목적지가 목적지 목록에 나타나는지 확인하십시오. 대상 이름 아래의 값은 대상 ID입니다.

대상 이름을클릭하여 대상을 편집합니다. 채널/페이지열의 링크를 클릭하여 다른 브라우저 탭에서 YouTube 채널을 엽니다.

YouTube 기본값 설정

소셜 플랫폼에 동영상을게시할 때 대상 기본값이 기본 비디오 속성으로 사용됩니다 . YouTube 대상에 대한 게시 기본값을 구성하려면 대상 이름을 클릭한 다음편집하다에서이 대상의 기본값부분. 다음 비디오 속성을 구성 할 수 있습니다.

  • 게시물 정보
  • 시간표
  • 캡션
  • 시계
  • 특허

게시물 정보 속성 구성

게시물 정보링크를 클릭합니다. 다음 YouTube 동영상 속성을 설정할 수 있습니다.

  • 기술 -Video Cloud 짧거나 긴 설명을 사용하거나 사용할 기본 설명을 입력하는 옵션
  • 원래 언어 코드 -번역 된 동영상 제목과 설명을 동영상에 추가하는 데 사용됩니다. 2 자리 언어 코드를 입력하고 목록에서 선택하세요.
  • YouTube 카테고리 -동영상 콘텐츠 카테고리
  • 태그 -Video Cloud 태그 포함 또는 제외 옵션
  • Video Cloud 메타 데이터를 YouTube 태그로 포함 -동영상 ID와 참조 ID를 YouTube 태그로 포함하는 옵션
  • 추가 태그 -동영상에 추가 태그를 입력하는 옵션
  • 썸네일 소스 -YouTube 썸네일, Video Cloud 썸네일을 사용하거나 자신의 썸네일 이미지를 업로드하는 옵션

일정 속성 구성

일정 (Schedule)링크를 클릭하고 일정 설정을 선택합니다.

  • 즉시 게시 -동영상을 즉시 게시하려면
  • Video Cloud 시작일 사용 -비디오에 설정된 Video Cloud 시작 날짜를 사용합니다. 이 경우 YouTube는 비디오 클라우드에' 지정된 '예정된 시작 날짜를 기준으로 동영상을 공개하도록 유도합니다. 해당 날짜가 과거 인 경우 YouTube는 동영상을 즉시 공개합니다.
참고: 비디오 가시성을미등록또는비공개로설정하려면일정옵션을 설정합니다.를 사용하여 즉시게시하고가시성옵션을미등록또는비공개로 설정합니다 .

캡션 속성 구성

캡션링크를 클릭하고 다음 옵션을 선택합니다.

  • Video Cloud 캡션 사용 -소셜 모듈은 동영상과 관련된 캡션을 YouTube에 배포하려고합니다. Social 모듈은 어디에 있든 캡션 파일에 액세스하려고 시도합니다. 원격 URL을 사용하여 캡션 파일의 위치를 지정하고 파일에 액세스 할 수있는 경우 (예: 우리가 가져올 수없는 방식으로 지리적 제한되지 않음) Social은 해당 파일을 YouTube에 배포합니다.
  • 캡션 없음 -캡션이 사용되지 않습니다.
참고: Video Cloud에서 캡션 작업에 대한 자세한 내용은개요를 참조하십시오. 비디오에 캡션 추가 .

가시성 속성 구성

가시성링크를 클릭하고 동영상의 개인 정보 설정을 선택합니다. 다음 개인 정보 설정을 사용할 수 있습니다.

  • 공공의 -누구나 동영상을 볼 수 있습니다.
  • 미등록 -동영상 링크가있는 사람은 누구나 볼 수 있습니다.
  • 은밀한 -비디오는 귀하와 귀하가 선택한 사용자 만 볼 수 있습니다.
  • YT_Privacy_Status 맞춤 입력란 사용 -비디오와 관련된 사용자 정의 필드를 사용하여 개인 정보 설정을 동적으로 할당합니다. 자세한 내용은 동영상의 개인 정보 상태설정을 참조하십시오 .

동영상이 YouTube에 게시될 때 구독자에게알리는 설정도있습니다. 선택하면 구독자는 구독 피드에서 동영상을 볼 수 있으며 동영상이 업로드되었다는 알림을받을 수 있습니다.

참고: YouTube 개인 정보 설정에 대한 자세한 내용은 YouTube 설명서를 참조하십시오 .

라이선스 속성 구성

라이선스링크를 클릭하고 비디오에 대한 라이선스 및 권한 소유권 설정을 선택합니다. 다음 설정을 사용할 수 있습니다.

  • 표준 YouTube 라이선스
  • 크리에이티브 커먼즈-저작자 표시
참고: YouTube 라이선스 및 권한 소유권에 대한 자세한 내용은 YouTube 설명서를 참조하십시오 .

Facebook 목적지 만들기

Facebook 페이지에 비디오를 게시하려는 경우 Facebook 대상을 만듭니다. 비디오는 개인 프로필 페이지가 아닌 Facebook 페이지에만 배포 할 수 있습니다. Facebook 대상을 만들려면 다음 단계를 따르세요.

  1. Brightcove Social을 엽니 다.
  2. 딸깍 하는 소리목적지왼쪽 탐색에서
  3. 대상추가를 클릭합니다 .
  4. 페이스북으로로그인을 클릭합니다 .
  5. Facebook 계정에 로그인하라는 대화 상자가 열립니다. 자격 증명을 입력하고로그인을 클릭합니다 .
  6. Brightcove Social에서 사용자의 프로필 정보를 수신 할 것임을 확인하는 대화 상자가 열립니다. 계속을 클릭합니다 .
  7. Brightcove Social이 페이지를 관리 및 게시하고 페이지 및 앱 인사이트에 액세스 할 수 있음을 확인하는 대화 상자가 열립니다. 확인 을 클릭합니다 .
  8. 목적지에 고유하고 설명이 포함 된 이름을 지정하십시오. Facebook 페이지가 여러 개인 경우 드롭다운 목록을 사용하여 게시할 페이지를 선택한 다음추가를 클릭합니다 .
    참고: 개인 프로필 페이지에는 동영상을 게시할 수 없습니다.
  9. 기본 정보와 목적지 기본값이 표시됩니다. 기본 정보섹션에서편집버튼을 클릭하여 대상 이름을 편집합니다. 기본값 구성에 대한 자세한 내용은 Facebook 기본값설정섹션을 참조하십시오.
  10. 딸깍 하는 소리목적지을 클릭하고 목적지가 목적지 목록에 나타나는지 확인하십시오. 목적지 이름 아래의 값은 목적지 ID입니다.

목적지를 편집하려면 목적지 이름을 클릭하십시오. 채널 / 페이지 열의 링크를 클릭하여 다른 브라우저 탭에서 Facebook 페이지를 엽니 다.

Facebook 기본값 설정

소셜 플랫폼에 동영상을게시할 때 대상 기본값이 기본 비디오 속성으로 사용됩니다 . Facebook 대상에 대한 게시 기본값을 구성하려면 대상 이름을 클릭한 다음편집하다에서이 대상의 기본값부분. 다음 Facebook 비디오 속성을 구성 할 수 있습니다.

  • 게시물 정보
  • 시간표
  • 캡션
  • 표적
  • 수익 창출
  • 진상
  • 분포
  • 추적

게시물 정보 속성 구성

게시물 정보링크를 클릭합니다. 다음 Facebook 비디오 속성을 설정할 수 있습니다.

  • 상태 메시지 -Video Cloud 짧거나 긴 설명을 사용하거나 사용할 기본 설명을 입력하는 옵션
  • 태그 -사람들이 내 동영상을 찾는 데 도움이되는 태그
  • Facebook 카테고리 -동영상 콘텐츠 카테고리
  • 썸네일 소스 -Facebook 썸네일, Video Cloud 썸네일 사용 또는 자신의 썸네일 이미지 업로드 옵션

일정 속성 구성

일정링크를 클릭합니다. 선택하다즉시 게시비디오를 즉시 게시하거나 Video Cloud 시작일 사용동영상에 설정된 시작 날짜를 사용합니다. Video Cloud 종료일에 게시물이만료되지않거나게시물이 만료되도록 할 수도있습니다.

캡션 속성 구성

캡션링크를 클릭합니다. 선택하다 Video Cloud 캡션 사용비디오와 연결된 캡션을 사용하거나캡션 없음캡션을 사용하지 않습니다.

참고: Video Cloud에서 캡션 작업에 대한 자세한 내용은개요를 참조하십시오. 비디오에 캡션 추가 .

대상 속성 구성

Target링크를 클릭합니다. 다음 속성을 설정할 수 있습니다.

  • 잠재 고객 최적화 관심사 -게시물을 즐길 가능성이 가장 높은 사람들의 관심사를 설명하는 태그를 추가합니다. 자세한 내용은 Facebook 설명서를 참조하십시오 .
  • 위치 제한 -게시물보기에 포함하거나 제외 할 국가 추가

수익 창출 속성 구성

수익 창출링크를 클릭합니다. 다음 속성을 설정할 수 있습니다.

  • 이 목적지에 대한 게시물의 비즈니스 파트너는 누구입니까? -브랜디 드 콘텐츠 게시물에 대한 비즈니스 파트너를 추가합니다. 입력하는 동안 일치하는 비즈니스 목록이 표시됩니다. Facebook 브랜드 콘텐츠에 대한 자세한 내용은 Facebook 설명서를 참조하십시오 .
  • 비즈니스 파트너가 게시물을 높일 수 있도록 허용

권한 속성 구성

참고: 권한 옵션이 나타나려면 Facebook 계정에 Facebook 권한 관리자에 대한 액세스 권한이 있어야 합니다.

권한링크를 클릭합니다. 다음 속성을 설정할 수 있습니다.

  • 일치하는 콘텐츠 감지 -Facebook 권한 관리를 활성화하려면 선택하십시오. 자세한 내용은 Facebook 설명서를 참조하십시오 .
  • 소유권 지정 -비디오, 오디오 또는 둘 다에 대한 저작권 소유권 지정
  • 국가 선택 - 전 세계 , 국가포함또는 국가제외를 선택합니다 . 포함 또는 제외를 선택한 경우위치 제한을 추가합니다 .
  • 일치 규칙 적용 -규칙을 선택합니다. 일치 규칙은사용자가 일치하는 동영상을 업로드할 때 발생하는 자동 동작을 결정합니다. 경기 규칙은 Facebook 내부에서 관리됩니다.
  • 권한 부여 -귀하의 콘텐츠와 일치하는 동영상을 게시 할 수 있도록 특정 사람, 페이지 또는 Instagram 계정을 승인합니다.

배포 속성 구성

배포링크를 클릭하여 비디오를 볼 수 있는 위치를 설정합니다. 다음과 같은사용자 지정 배포 및 중재옵션을 사용할 수 있습니다.

  • 뉴스 피드에 표시 -뉴스 피드에 비디오를 보여줍니다
  • 퍼가기 허용 -타사 사이트에 동영상 삽입 허용
  • 비밀 비디오로 추가 -동영상은 직접 URL을 통해서만 액세스 할 수 있으며 Facebook 사용자는 검색 할 수 없습니다.
    • 반응, 댓글 및 공유 끄기 -좋아요 및 댓글 허용 안함

추적 속성 구성

추적링크를 클릭하여 시스템 전체에서 비디오를 추적하는 범용 ID를 할당합니다. 다음과 같은범용 ID옵션을 사용할 수 있습니다.

  • 백지로 둬
  • Video Cloud에서 비디오 ID 사용
  • Video Cloud의 참조 ID 사용 (있는 경우)

Twitter 피드에 비디오를 게시하려는 경우 Twitter 대상을 만듭니다. Twitter 대상을 만들려면 다음 단계를 따르세요.

  1. Brightcove Social을 엽니 다.
  2. 딸깍 하는 소리목적지왼쪽 탐색에서
  3. 대상추가를 클릭합니다 .
  4. 트위터로로그인을 선택합니다 .
  5. Twitter 계정에 로그인하고 Brightcove Social에서 계정을 사용하도록 승인하라는 메시지가 표시됩니다. 비디오를 게시 할 계정에 로그인하십시오. 로그인하라는 메시지가 표시되지 않으면 Twitter 계정에 로그인 한 것일 수 있습니다. 계속하기 전에 올바른 계정으로 로그인했는지 확인하십시오. 앱권한 부여를 클릭합니다 .
  6. 목적지에 고유하고 설명이 포함 된 이름을 지정하십시오. 추가를 클릭합니다 .
  7. 기본 정보와 목적지 기본값이 표시됩니다. 기본 정보섹션에서편집버튼을 클릭하여 대상 이름을 편집합니다. 기본값 구성에 대한 자세한 내용은 Twitter 기본값설정섹션을 참조하십시오.
  8. 딸깍 하는 소리목적지을 클릭하고 목적지가 목적지 목록에 나타나는지 확인하십시오. 목적지 이름 아래의 값은 목적지 ID입니다.

목적지를 편집하려면 목적지 이름을 클릭하십시오. 채널 / 페이지 열의 링크를 클릭하여 다른 브라우저 탭에서 Twitter 계정을 엽니 다.

Twitter 기본값 설정

소셜 플랫폼에 동영상을게시할 때 대상 기본값이 기본 비디오 속성으로 사용됩니다 . Twitter 대상에 대한 게시 기본값을 구성하려면 대상 이름을 클릭한 다음편집하다에서이 대상의 기본값부분.

기본트윗 콘텐츠 구성 :

  • Video Cloud에서 비디오 이름 사용
  • Video Cloud에서 TW_Tweet 사용자 정의 필드 사용
  • 맞춤 기본 트윗 설정

목적지 삭제

목적지를 삭제하려면 다음 단계를 따르십시오.

  1. Brightcove Social을 엽니 다.
  2. 딸깍 하는 소리목적지왼쪽 탐색에서 목적지 목록이 나타납니다.
  3. 삭제할 목적지 이름을 클릭하십시오.
  4. 목적지 세부 정보가 표시됩니다. 삭제 아이콘() 클릭삭제삭제를 확인합니다.
참고: 대상을 사용하는 자동 동기화 작업이 생성 된 경우 삭제 아이콘이 나타나지 않습니다. 대상을 삭제하기 전에 자동 동기화 작업을 제거해야 합니다.

목적지에 다시 연결

Brightcove Social은 대상에 문제가있는 경우 오류 메시지를 표시합니다.

오류 아이콘()이 수신인 이름 옆에 표시됩니다. 이는 Brightcove 권한이 소셜 미디어 계정에서 제거되거나 대상을 생성 할 때 사용 된 자격 증명이 더 이상 유효하지 않은 경우 발생할 수 있습니다. 이 문제를 해결하려면 다음과 같이 하십시오.

  1. 목적지 이름을 클릭하십시오.
  2. 페이지 상단의재연결링크를 클릭합니다.
  3. 소셜 미디어 계정으로 다시 인증하라는 메시지가 표시 될 수 있습니다.
