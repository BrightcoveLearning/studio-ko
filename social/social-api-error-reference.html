상태 이름 메시지 원인

소셜 공유 오류

소셜 공유 오류는 API 요청에 대한 별도의 오류 응답으로 반환되지 않고 일반 응답의errors배열에서 반환됩니다.

공유 작업이 실패하면 API는 다음을 포함하는 오류 객체를"result": "ERROR"보고하고 반환합니다.

  • error_code -반환 된 오류 코드는 서버가 소셜 플랫폼에서 기록합니다 (있는 경우).
  • error_message -서버에 의해 기록되거나 소셜 플랫폼에서 반환 된 오류 메시지 (있는 경우).
  • platform_error -소셜 플랫폼에서 반환 된 오류 개체의 JSON 표현 (있는 경우).

다음은 예입니다.

"errors": [
  {
    "error_code": "UPLOAD_FAILURE",
    "message": "Upload error: Received Facebook error response of type OAuthException: (#100) sponsor_id requires publishing profile/page to be verified or branding partner. (code 100, subcode null)"
  }
]

반환되는 오류에 대한 자세한 내용은 특정 소셜 플랫폼에 대한 설명서를 참조하십시오.
