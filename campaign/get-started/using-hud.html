헤드 업 디스플레이 Chrome 확장 프로그램 사용

{{ page.description }}

HUD (헤드 업 디스플레이)는 다음 기능을 제공하는 Chrome 확장 프로그램입니다.

  • 라이브 페이지에서 비디오의 성능 메트릭보기
  • 계정에서 동영상 게시, 편집 및 삭제

Chrome 확장 프로그램 설치

헤드 업 디스플레이 Chrome 확장 프로그램을 설치하려면 다음 단계를 따르세요.

  1. Chrome 웹 브라우저를 사용하여여기에서 브라이트코브 캠페인 확장 페이지에액세스합니다.
  2. Chrome에추가를 클릭합니다 .
  3. 메시지가 나타나면 확장 프로그램추가를 클릭합니다 .
  4. 주소 표시 줄 옆에있는 Brightcove 로고를 찾아 확장이 설치되었는지 확인합니다.

헤드 업 디스플레이 열기

헤드 업 디스플레이는 비디오 분석을보고 계정에서 비디오를 게시, 편집 및 삭제하는 데 유용합니다. HUD를 열려면 다음 단계를 따르십시오.

  1. Google 크롬을 엽니 다.
  2. 주소 표시 줄 옆에있는 Brightcove 로고를 클릭하여 확장 프로그램을 엽니 다.
  3. Brightcove Campaign 응용 프로그램에 활성 세션이없는 경우 로그인하라는 메시지가 표시됩니다. 이메일 주소와 비밀번호를 입력하세요. 로그인을 클릭합니다 .

HUD 상단의 아이콘을 사용하여 다음을 수행 할 수 있습니다.

  • 분석 검토
  • 비디오 관리

웹으로 이동을 클릭하면 브라이트코브 캠페인 애플리케이션이 브라우저 창에 열립니다.

계정 선택

HUD의 오른쪽 상단에 현재 캠페인 계정이 표시됩니다.

여러 캠페인 계정에 액세스 할 수있는 경우 계정 이름을 클릭하면 계정 선택기가 열립니다. 계정 이름드롭다운을 클릭하여 작업할 다른 계정을 선택합니다.

HUD에서 로그아웃하려면로그아웃을 클릭합니다 .

비디오 분석 검토

HUD가 처음 열리면 비디오 분석이 표시됩니다. 분석 아이콘() 분석 모드로 이동합니다. HUD는 Brightcove Campaign ™을 사용하여 게시 된 페이지의 비디오에 대한 분석 만 표시합니다.

HUD는 페이지의 동영상에 대한 기본 정보를 표시합니다.

  • 비디오 이름
  • 비디오 ID
  • 비디오 콘텐츠 유형
  • 만든 날짜

페이지에 동영상이 여러 개 있는 경우이전 비디오및다음 비디오버튼이 활성화됩니다.

HUD의 비디오 분석은 "실시간에 가깝습니다". 분석은 일반적으로 몇 분 정도 지연되지만 트래픽이 급증하는 동안 지연이 늘어날 수 있습니다. 지난 30 일 동안의 다음 분석 데이터가 표시됩니다.

  • 비디오 노출 -동영상이로드되고 재생할 준비가 된 횟수
  • 비디오 시작 -동영상 재생이 시작된 횟수
  • 평균 조회 시간 -총 조회 시간 / 동영상 시작
  • 비디오 벤치마크 점수 - 동영상의 실적을 측정하는 전체 평점 ( 자세히 알아보기... )
  • 재생률 -동영상 시작 / 동영상 노출
  • 평균 조회 율 -평균 조회 시간 / 동영상 길이

동영상에 대한 모든 분석을보려면 모든 분석 보기링크를 클릭합니다.

비디오 관리

HUD를 사용하여 계정에서 비디오를 관리하고 게시하려면 비디오 아이콘() . 계정의 동영상 목록이 나타납니다.

동영상 위로 마우스를 가져 가면 다음 기능을 제공하는 일련의 버튼이 표시됩니다.

  • 비디오 게시
  • 비디오 편집
  • 비디오 삭제

비디오 게시

비디오 위로 마우스를 가져가게시를 클릭합니다 . 대상, 웹 페이지또는이메일을 선택합니다 . 마법사는 소스 코드를 생성하거나 이메일 메시지에 사용할 비디오 썸네일을 가져 오는 과정을 안내합니다.

비디오 편집

비디오 위로 마우스를 가져가편집을 클릭합니다 . 필요에 따라 비디오 속성을 편집한 다음 [ 저장] 을 클릭합니다 .

비디오 삭제

비디오 위로 마우스를 가져가삭제 를 클릭합니다 . 삭제를 확인하는 메시지가 표시됩니다.
