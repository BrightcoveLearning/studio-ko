REST API를 사용하여 새로운 HubSpot 통합을 설정할 때 비디오 시청 데이터가 실제로 HubSpot에 전달되고 있는지 테스트하고 확인하는 것이 어려울 수 있습니다. 이 항목에서는 통합이 작동하고 데이터 및 리드보기가 HubSpot으로 전송되는지를 체계적으로 테스트하고 확인하는 방법을 배웁니다.

참고: 이 항목은 REST API를 사용하여 만들어진 HubSpot 연결을 확인하기 위한 것입니다.
참고: HubSpot에서 사용하도록 캠페인 모듈을 구성하는 방법에 대한 자세한 내용은 REST API를 사용하여 HubSpot과 비디오 클라우드 통합을 참조하십시오 .

시작하기 전에

통합을 검증하기 전에 다음이 있다고 가정합니다.

  • 성공적으로 완료 HubSpot과 Brightcove 캠페인 통합
  • 하나 이상 생성됨캠페인 지원 플레이어 HubSpot 연결을 사용하도록 구성했습니다.

테스트 리드 양식 구성 및 비디오 게시

테스트를 위해 Campaign에서 사용자 정의 리드 양식을 만들고 구성해야 합니다. 이것은 귀하가 Campaign에 붙여넣은 임베드 코드의 HubSpot 양식입니다. 프로덕션 환경에서 리드 양식을 사용하지 않더라도이 리드 양식은 사용자가 데이터베이스와 페이지에 알려져 있는지 확인하는 데 사용됩니다. HubSpot 양식을 만들고 이를 사용하도록 Campaign을 구성하려면 다음 단계를 따르십시오.

  1. HubSpot 계정에 로그인하십시오.
  2. HubSpot 양식을 만들고 스타일을 지정합니다. HubSpot 양식 생성에 대한 자세한 내용은 HubSpot용 사용자 지정 리드 양식만들기를 참조하십시오 .
  3. 딸깍 하는 소리공유하다페이지 상단에 있습니다.
  4. 양식 소스 코드를 클립 보드에 복사합니다.
  5. 캠페인 모듈로 돌아갑니다.
  6. HubSpot 양식 포함 코드를 사용하여 새 리드 양식을 만듭니다.
  7. 5 초 표시에 양식을 표시하도록 사용자 정의 양식을 설정하십시오. 이렇게하면 언제 표시되어야하는지 정확히 알 수 있습니다.
  8. 리드가 이미 알려진 경우에도항상 가망 고객 양식 표시 확인란을선택합니다. 이렇게하면 테스트 중에 양식을 작성한 후에도 양식이 계속 표시됩니다.
  9. 양식을 저장합니다.
  10. 방금 생성 된 HubSpot 리드 양식을 사용하도록 HubSpot 연결로 Brightcove Player를 구성합니다.

비디오를보기위한 미리보기 링크 생성

비디오에 대한 미리보기 링크를 생성하려면 다음 단계를 따르십시오.

  1. 열기미디어기준 치수.
  2. 비디오를 선택하고 웹 플레이어에 게시합니다. 리드 양식이 있는 캠페인 지원 플레이어를 선택해야 합니다.
  3. 미리보기 링크를 클릭하여 새 브라우저 창에서 비디오를 엽니 다. 이러한 방식으로 테스트하면 HTML 페이지에 포함 된 플레이어가 도입 할 수있는 모든 잠재적 변수가 제거됩니다.
  4. 비디오가 재생되고 리드 양식이 5 초 표시에 표시되는지 확인합니다.
  5. HubSpot에서 나중에 쉽게 찾을 수 있도록 고유 연락처 정보와 이메일 주소 (예: 유명인 이름 또는 “bctest”라는 단어가 포함 된 이름) 를 사용하여 리드 양식을 작성하십시오. 이렇게하면 브라우저, 플레이어 또는 데이터베이스 수준 변수가 제거됩니다. 새 사용자로 양식을 작성하면 이제 HubSpot에서 알려진 사용자임을 확인할 수 있습니다.

캠페인이 데이터를 성공적으로 캡처하고 있는지 확인

HubSpot에서 비디오 보기 데이터를 찾기 전에 먼저 Campaign이 실제로 비디오 보기를 캡처했는지 확인하십시오. 데이터 보기가 캠페인 UI에 표시되는 데 최대 3시간이 소요될 수 있습니다. Campaign이 데이터를 성공적으로 캡처하고 있는지 확인하려면 다음 단계를 따르십시오.

  1. 캠페인 모듈을 엽니다.
  2. 딸깍 하는 소리모든 최근 활동왼쪽 탐색에서
  3. 방금 시청 한 동영상의 이름을 보여주는 최근 타임 스탬프가있는 이벤트가 있는지 확인합니다. 이벤트 상태는처리되지 않아야합니다. 이는 Campaign이 보기를 캡처했지만 아직 HubSpot에 동기화를 시도하지 않았음을 나타냅니다.
  4. Campaign과 HubSpot 간의 동기화를 수동으로 트리거하려면동기화 활동왼쪽 탐색에서
  5. 연결선택드롭다운 목록에서 HubSpot 통합을 선택합니다. 동기화테스트버튼이 나타납니다.
  6. 동기화테스트를 클릭하여 수동 동기화를 트리거합니다. 수동 동기화를 트리거 할 수있는 빈도로 제한됩니다. 기본적으로 Campaign은 한 시간에 한 번씩 HubSpot에 보기를 동기화합니다.
  7. 딸깍 하는 소리모든 최근 활동왼쪽 탐색에서
  8. 뷰 이벤트의 상태가대기열에 표시됩니다 . 데이터가 실제로 HubSpot에 동기화되는 데 15 분 정도 걸릴 수 있습니다. 데이터가 동기화되면 상태는 다음 두 가지 상태 중 하나로 변경됩니다.
    • 동기화 됨 -사용자가 알려진 사용자였으며 데이터가 활동 로그에 성공적으로 전달되었음을 나타냅니다. 데이터는 또한 몇 분 내에 HubSpot에서 볼 수 있어야합니다.
    • unknown_contact -사용자가 알 수없는 것으로 간주되어 HubSpot이 데이터를 수락 할 수 없음을 나타냅니다. unknown_contact상태가 표시되면 올바른 플레이어를 사용하여 비디오를 게시하고 HubSpot 양식을 마련하고 페이지에 포함되지 않은 미리 보기 링크를 사용하여 테스트하고 있는지 확인하십시오.

비디오보기 데이터가 HubSpot에 있는지 확인

캠페인이 표시되면동기화상태에서 데이터가 HubSpot에 있는지 확인하려면 다음 단계를 따르십시오.

참고: HubSpot UI가 Campaign에서 받은 새 데이터로 업데이트하는 데 5~10분 정도 걸립니다.
  1. HubSpot 계정에 로그인하십시오.
  2. HubSpot에서연락처영역으로 이동하여 리드 양식을 작성할 때 사용한 고유 이름 또는 이메일 주소를 찾습니다.
  3. 연락처를 클릭하여 활동 로그를 엽니 다.
  4. 클릭활동 필터링드롭다운(페이지 상단 아래활동메뉴)를 확인하고브라이트코브 비디오 커넥트확인됩니다. 이렇게하면 비디오보기 활동이 활동 로그에 표시됩니다.
  5. 활동 로그에서 다음을 찾습니다. 브라이트코브활동. 동영상 보기 이벤트에 대한 데이터를 보려면 세부정보를 확장합니다.

HubSpot은 API를 통해 전달되는 데이터를 알려진 연락처에 할당해야합니다. HubSpot은 알려진 연락처의 데이터 만 허용합니다. 알 수없는 연락처의 데이터는 동기화되지 않습니다. 리드 양식을 사용하지 않으면 알 수없는 사용자가 알려진 사용자로 변환 할 수있는 방법이 없으므로 데이터가 동기화되지 않습니다.
