보기 이벤트는 Campaign 지원 플레이어를 사용하여 비디오를 시청할 때 기록됩니다. 리드 데이터는 사용자가 Campaign 리드 양식에 데이터를 입력할 때 캡처됩니다. 캠페인 모듈은 두 가지 유형의 리드 양식을 제공합니다.

  • 캠페인 리드 양식 - 이 양식은 캠페인 모듈의 사전 정의된 필드 세트에서 선택하여 작성됩니다. 이러한 양식은 Video Cloud에 리드 데이터를 저장합니다. 캠페인 리드 양식 작성에 대한 자세한 내용은 다음을 참조하십시오. 캠페인 리드 양식 만들기 .
  • 맞춤형 리드 양식 -이러한 양식은 마케팅 자동화 플랫폼을 사용하여 작성됩니다. 그런 다음 양식 포함 코드를 캠페인 모듈에서 사용하여 사용자 정의 리드 양식을 만듭니다. 이러한 양식은 리드 데이터를 마케팅 자동화 플랫폼으로 보냅니다. 리드 양식 작성에 대한 정보는 마케팅 플랫폼에 대한 문서를 참조하십시오.
    • Eloqua
    • Marketo
    • 영업
    • HubSpot

캠페인 모듈에서 리드 데이터를 내보낼 때 캠페인 리드 양식을 사용하여 캡처한 데이터만 내보내집니다. 맞춤형 리드 양식의 리드 데이터는 마케팅 자동화 소프트웨어에 저장됩니다.

데이터를 내보내려면 다음 단계를 따르십시오.

  1. 캠페인 모듈을 엽니다.
  2. 딸깍 하는 소리수출 캠페인 데이터왼쪽 탐색에서
    참고: 데이터 내보내기 메뉴에 액세스하려면 계정 관리자여야 합니다.
  3. 선택데이터 연결에서 데이터를 내보낼 수 있습니다. 데이터 연결드롭다운에는 구성된 모든 통합이 나열됩니다.
  4. 내보낼 데이터를 선택하십시오.
    • 이벤트 보기 - 캠페인 지원 플레이어가 녹음
    • 리드 -리드 양식을 사용하여 캡처
  5. 날짜 선택기를 사용하여시작 날짜및종료 날짜를 선택합니다 .
  6.  내보내기 형식을 선택합니다 .
    • 쉼표로 구분 된 값 (CSV)
    • Excel (XLS)
  7. 내보내기를클릭하여 리드 데이터를 다운로드합니다.
  8. 리드 양식 및보기 이벤트 데이터는 별도의 파일로 다운로드됩니다. 가망 고객데이터에는 가망 고객 양식에 입력된 데이터와 다음 필드가 포함됩니다.
    • externalId -모든 최근 활동 페이지에 표시되는 외부 ID입니다. Oracle Eloqua의 경우 Eloqua guid입니다. Marketo와 함께라면 Marketo 가이드입니다.
    • playerId -양식이 표시 될 때 사용중인 플레이어의 플레이어 ID
    • videoId -폼이 표시 될 때 재생되는 비디오의 비디오 ID
    • pageURL -리드 양식을 표시 한 페이지 URL (갤러리 환경에만 해당)
    • createdAt -데이터가 캡처 된 날짜 및 시간
    • 경험 -데이터를 수집 한 갤러리 체험 (갤러리 체험 전용)

    이벤트데이터 보기에는 다음 필드가 포함됩니다.

    • externalId -모든 최근 활동 페이지에 표시되는 외부 ID입니다. Oracle Eloqua의 경우 Eloqua guid입니다. Marketo와 함께라면 Marketo 가이드입니다.
    • videoId -동영상의 동영상 ID
    • videoName -동영상 이름
    • 보았다 -비디오 시청률 (가장 가까운 백분율)
    • timeWatched -총 시청 시간
    • playerId -동영상 시청에 사용 된 플레이어의 플레이어 ID
    • pageUrl -플레이어가 포함 된 페이지의 URL
    • createdAt -데이터가 캡처 된 날짜 및 시간
    • UpdateDat - 이벤트가 마지막으로 저장된 날짜 및 시간. 시청자가 시청을 중단한 시점을 대략적으로 변환합니다. 동영상의 길이가 단 1분 이면 createDat 및 UpdateDat이 매우 가깝습니다.
    • isSynced -이보기 이벤트가 게시자의 마케팅 자동화 도구로 자동으로 내보내 졌는지 여부를 지정합니다.
    • 추적 ID -URL 매개 변수로 전달되는 추적 ID입니다. 추적 ID 작업에 대한 정보는 다음을 참조하십시오. 캠페인 모듈로 사용자 정의 통합 생성 .
    • UTM 필드 -utmSource, utmMedium, utmCampaign, utmTerm 및 utmContent. UTM 필드 작업에 대한 정보는 다음을 참조하십시오. 캠페인 모듈에서 UTM 필드 작업 .
    • metric1-3 , event1-3 - 사용자 지정 지표 및 이벤트를 사용할 때 사용됩니다.
