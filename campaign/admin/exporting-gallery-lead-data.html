갤러리 경험이 리드 양식을 사용하도록 구성된 경우 갤러리 리드 양식 데이터가 수집됩니다. 잠재 고객 데이터를 수집하도록 포털 환경을 구성하는 방법에 대한 자세한 내용은 포털 경험에 가망 고객 양식사용을 참조하십시오 .

캠페인 모듈에서 갤러리 리드를 내보낼 때 캠페인 리드 양식을 사용하여 캡처한 데이터만 내보내집니다. 맞춤형 리드 양식의 리드 데이터는 마케팅 자동화 소프트웨어에 저장됩니다.

참고: 관리자 권한이 있는 사용자만 리드 데이터를 다운로드할 수 있습니다.

갤러리 리드 데이터를 내보내려면 다음 단계를 따르세요.

  1. 열기운동기준 치수.
  2. 딸깍 하는 소리갤러리 리드 내보내기왼쪽 탐색에서
  3. 리드를 내보낼 갤러리 환경을 선택합니다.
  4. 가망 고객의시작 일자및종료일자를 입력합니다.
  5. 내보내기 형식, 쉼표로 구분된 값또는 Excel을 선택합니다 .
  6. 내보내기버튼을 클릭하여 리드 데이터를 다운로드합니다.

내 보낸 파일에는 다음 필드가 포함됩니다.

  • 가능한 모든 리드 양식 필드
    • 이메일 주소
    • 이름
    • lastName
    • 업무 폰
    • country
    • 회사 이름
    • 산업
  • externalId -시청자를 고유하게 식별하는 데 사용되는 ID
  • playerId -리드 양식을 표시 한 플레이어 ID
  • videoId -데이터 수집시 시청중인 영상의 영상 ID
  • pageUrl -데이터 수집시 페이지 URL 뷰어 켜짐
  • createdAt -수집 된 날짜와 시간
  • 경험 -데이터 수집시 표시되는 체험 명
