뷰어 데이터 관리 옵션을 사용하면 사용자를 검색 한 다음 해당 사용자를 제거하거나 해당 사용자의 데이터를 내보낼 수 있습니다.

메모:
  • 일반적으로 마케팅 자동화 플랫폼(MAP 통합)과 통합할 때 데이터 보유 기간 설정은 데이터 보유 기간을 제어하는 데 사용됩니다.
  • 뷰어 데이터 관리 메뉴에 액세스하려면 계정 관리자여야 합니다.

뷰어 데이터를 관리하려면 다음 단계를 따르십시오.

  1. 캠페인 모듈을 엽니다.
  2. 딸깍 하는 소리뷰어 데이터 관리왼쪽 탐색에서
  3. 검색 할 사용자를 입력하십시오. 일반적으로이 값은 다음 값 중 하나입니다.
    • 이메일 - 캠페인 추적을 사용하고 캠페인 리드 양식을 사용하거나 SSO 통합과 함께 사용하는 경우
    • 사용자 ID -SSO 또는 사용자 정의 통합을 사용하는 경우
    • 사용자 지정 GUID -익명 사용자의 경우
    • 외부 GUID -MAP 통합을 사용하는 경우
  4. 일치하는 항목이 있으면 작업을 선택하십시오.
    • 뷰어 데이터 삭제 -사용자에 대한보기 및 리드 데이터 제거
    • 뷰어 데이터 내보내기 -리드를 내보내고 사용자에 대한 이벤트 데이터를 봅니다. 리드 데이터에는 캠페인 리드 양식에 입력된 데이터가 포함됩니다. 보기 이벤트 데이터에는 뷰어에 대한 모든보기 이벤트가 포함됩니다.
