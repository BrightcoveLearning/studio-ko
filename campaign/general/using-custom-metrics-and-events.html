캠페인 플레이어 플러그인에는 이벤트를 보기 위해 사용자 정의 이벤트 및 메트릭을 첨부하기 위한 클라이언트 측 API가 포함되어 있습니다. 다음 예제는 사용자 지정 이벤트 및 메트릭을 추적하는 방법을 설명합니다.

  1. Brightcove Player에 대한 참조를 얻습니다. 이 예제에서는id의myPlayerID가 플레이어 임베드 코드에 추가되었다고 가정합니다. 
      
var player = videojs(‘myPlayerID’);
  2. 오디언스 레디 이벤트를audience:ready들어보세요. 
      
player.on(‘audience:ready’, function() {
  // player.audience is now available for use
});
  3. 문자열 값을 추적하려면 를player.audience.track(‘event[1-3]’, ‘value’)호출합니다. 숫자 값을 추적하려면 를player.audience.track(‘metric[1-3]’, value)호출합니다. 반환 값은 이벤트가 성공적으로 전송되었는지 여부를 나타내는 부울입니다. 
      
// Store the string value “foo” in the slot event1
player.audience.track(‘event1’, ‘foo’);

// Store the string value “bar” in the slot event2
player.audience.track(‘event2’, ‘bar’);

// Store the decimal value 15.08 in slot metric1
player.audience.track(‘metric1’, 15.08);

// Replace the value in slot metric1 with the number 10,
// and validate that the call was sent successfully
var result = player.audience.track(‘metric1’, 10);

if (result) {
   console.log(‘metric sent successfully’);
}

마케팅 자동화 플랫폼과의 동기화

필드가 사용자 지정 활동 또는 CDO에 정의 된 경우보기 이벤트에 저장된 사용자 지정 이벤트 / 메트릭이 Marketo REST 및 Eloqua에 대해 동기화됩니다. Campaign은 또한 정의된 경우 HubSpot에 사용자 정의 이벤트/메트릭을 동기화하고 통합이 사용 중인 경우 Salesforce에 동기화합니다. 매핑 유형 NS 고급의이벤트/메트릭이 무언가에 매핑됩니다.

Marketo REST

Marketo REST 통합의 경우 게시자는 Marketo 의브라이트코브 비디오 보기사용자 지정 활동에 사용자 지정 이벤트 및 메트릭 필드가 있는지 확인해야 합니다. 사용자 지정 활동은 아래에 있습니다. 관리자 > 데이터베이스 관리 .

필드가 브라이트코브 비디오 보기 개체에 없는 경우:

  1. 브라이트코브 비디오 보기사용자 지정 활동을 업데이트하여 필드를 추가하거나
  2. Marketo 지원에 문의하여 필드를 추가하도록 요청하십시오.
  3. 필드를 추가한 후 캠페인으로 돌아가서업데이트 완료버튼을 클릭하면 Campaign이 새로 추가된 필드에 액세스할 수 있습니다.

Oracle Eloqua

Eloqua 통합의 경우 게시자는 BrightCoveVideoView사용자 지정 개체에 사용자 지정 이벤트 및 메트릭 필드가 있는지 확인해야 합니다.

필드가 없으면 다음 중 하나를 사용하여 필드를 BrightCoveVideoView개체에 추가해야 합니다.

  1. Eloqua 계정 다시 연결 또는
  2. 데이터 매핑 업그레이드 버튼을 사용합니다.

영업

Salesforce의 경우 캠페인 연결에서 다음을 사용해야 합니다. 매핑 유형 NS 고급의 . 이벤트및지표필드는 Salesforce 객체에 매핑할 수 있습니다.

메모

  • 이벤트 슬롯 값의 문자열 길이 제한은 250 자입니다.
  • 단일보기 세션 중에 여러 값이 동일한 이벤트 또는 메트릭 슬롯으로 전송되면 슬롯의 기존 값이 새 값으로 바뀝니다. 다음 예에서 event1에 저장된 최종 값은 "click3"입니다. 
      
player.audience.track(‘event1’, ‘click1’);
player.audience.track(‘event1’, ‘click2’);
player.audience.track(‘event1’, ‘click3’);
  • 세션에 대한보기 이벤트가 아직 존재하지 않으면 하나가 생성됩니다.
  • 의 결과라면track()이벤트가 전송되지 않았음을 나타내는 false인 경우 다음을 수행할 수 있습니다. 자세한 옵션을 활성화캠페인 플러그인에서 특정 이유와 함께 콘솔 로깅을 보려면

이벤트 및 메트릭보기

사용자 정의 이벤트 및 측정항목은 캠페인 UI에 표시됩니다. 왼쪽 탐색에서모든 최근 활동을 클릭합니다 . 사용자 지정 이벤트 또는 지표가 기록된 경우 상호 작용 이벤트표시링크가 활성화됩니다. 링크를 클릭하면 이벤트 / 메트릭 데이터가 표시됩니다.
