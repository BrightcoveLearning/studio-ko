캠페인 추적 통합을 사용할 때 뷰어를 식별하는 데 사용된 데이터를 기반으로 뷰어 프로필이 생성됩니다. 뷰어 프로필을 사용하면 지정된 뷰어에 대한보기 데이터를 볼 수 있습니다.

캠페인 지원 플레이어의 모든 동영상 조회수가 추적됩니다. 비디오 활동을 보려면뷰어 프로필(MAP 외)왼쪽 탐색에서 뷰어 프로필 페이지에는 선택한 기간 동안 캠페인 추적 통합의 모든 보기 활동 목록이 표시됩니다. 달력 컨트롤을 사용하여 새 날짜 범위를 선택할 수 있습니다. 뷰어 값은 사용자를 식별하는 데 사용 된 값입니다. 일반적으로 이는 리드 양식에 캡처 된 이메일 주소입니다. 열 머리글을 클릭하여 데이터를 정렬 할 수 있습니다.

참고: 보기 데이터가 나타나기까지 몇 시간 지연이 발생합니다.

뷰어에 대한 링크를 클릭하면 해당 뷰어에 대한 모든보기 활동이 표시됩니다. 열 머리글을 클릭하여 데이터를 정렬 할 수 있습니다.

페이지 상단의 CSV또는 XLS버튼을 클릭하여 데이터를 내보낼 수 있습니다.

기타 데이터보기 옵션

다른 방법으로 캠페인 데이터를 보거나 내보낼 수 있습니다.

  • 최근 활동 -지난 24 시간 동안의 영상보기 이벤트 데이터를보기 위해 사용합니다. 자세한 내용은 다음을 참조하십시오. 캠페인 모듈에서 플레이어 이벤트 보기 .
  • 비디오 활동 -동영상별로 시청자 활동을 볼 때 사용합니다. 자세한 내용은 비디오 활동보기를 참조하십시오 .
  • 데이터 내보내기 -비디오 뷰 및 리드 데이터 내보내기에 사용됩니다. 자세한 내용은 다음을 참조하십시오. 캠페인 모듈에서 캠페인 데이터 내보내기 .
  • Audience API -Audience API를 사용하여 비디오보기 및 리드 데이터를 프로그래밍 방식으로 검색 할 수 있습니다. 자세한 내용은개요를 참조하십시오. 오디언스 API .
