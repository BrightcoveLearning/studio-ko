Brightcove Campaign은 통합 플랫폼의 특정 필드에 일련의 비디오 시청 데이터를 제공합니다. 다음은 해당 필드의 예와 각 필드를 채울 데이터에 대한 설명입니다.

표준 필드

이 필드는 항상 Brightcove Campaign에 의해 채워집니다.

들 기술
video_id

시청 이벤트와 관련된 Brightcove 비디오의 ID
video_name

Video Cloud의 비디오 이름
추적 ID 추가 할 수 있습니다. 맞춤 추적 ID URL에
external_id 사용자를 식별하는 데 사용되는 글로벌 사용자 ID (GUID)
player_id 보기 이벤트를 생성 한 Brightcove Player의 ID
page_url 동영상을 본 페이지의 URL
보았다 시청률-시청 한 타임 라인의 아래까지
time_watched 동영상을 시청 한 시간 (초)-해당 동영상에 실제로 보낸 총 시간
created_at 제작 일
updated_at 마지막 업데이트 날짜
is_synced 뷰 이벤트가 타사 플랫폼에 동기화되었는지 여부를 나타내는 부울입니다.

추가 필드

이러한 필드는 자동으로 채워지지 않습니다. 추가 사용자 지정 데이터를추적하는 데 사용됩니다 . 기본적으로 Gallery In-Page에서 상호 작용을 캡처하도록 구성되어있습니다. 대화 형 기능을 사용하지 않는 경우이 필드는 비어 있습니다.

들 기술
event_1, event_2, event_3

맞춤 이벤트
metric_1, metric_2, metric_3

맞춤 측정 항목
UTM 필드 -utmSource, utmMedium, utmCampaign, utmTerm 및 utmContent UTM 필드 작업에 대한 정보는 다음을 참조하십시오. 캠페인 모듈에서 UTM 필드 작업 .

사용자 정의 필드

이러한 필드는 자동으로 채워지지 않습니다. 이러한 필드를 통해 참조 ID, 태그 또는 사용자 정의 필드를 MAP 또는 CRM에 동기화하는 옵션이 있습니다. 이것을 설정하려면연락처 지원예를 들어 다음과 같이 요청합니다. "내 Video Cloud 필드 "태그"를 캠페인 "custom_1" 필드에 매핑하십시오."

들 기술
custom_1, custom_2, custom_3
