UTM 코드 (UTM 매개 변수라고도 함)는 웹 콘텐츠의 성공을 추적하는 데 도움이되도록 URL에 추가 할 수있는 간단한 텍스트 스 니펫입니다. 모든 URL에 추가 할 수있는 5 가지 표준 매개 변수가 있습니다.

  • 운동
  • 출처
  • 매질
  • 문맥
  • 기간

UTM 코드는 일반적으로 맞춤 URL에 추가됩니다. 다음은 URL에 추가 된 UTM 코드 (굵게 표시됨)의 예입니다.

https://www.brightcove.com/en/blog/2018/08/define-your-video-strategy-3-simple-steps-1?utm_campaign=blogpost&utm_medium=social&utm_source=LinkedIn

UTM 코드를 사용하면 각 캠페인에 대한 사용자 지정 랜딩 페이지를 만들지 않고도 블로그 게시물, 인터넷 광고 또는 이메일 폭발이 어떻게 작동하는지 추적 할 수 있습니다. 블로그 게시물, 광고 및 이메일에 대한 고유 한 UTM 코드를 생성하여 더 많은 트래픽, 전환 등을 생성하는 데이터를 얻을 수 있습니다.

UTM 필드와 함께 작동하도록 데이터 연결 구성

Campaign은 URL 인수에서 UTM 필드를 구문 분석합니다. UTM 데이터는 마케팅 자동화 소프트웨어에서 또는 Audience API를 사용하여 액세스 할 수 있습니다. 필요한 캠페인 구성은 사용 중인 마케팅 자동화 플랫폼을 기반으로 합니다.

  • HubSpot
  • 영업
  • Eloqua
  • Marketo

HubSpot

HubSpot 데이터 연결을 사용하면 추가 설정이 필요하지 않습니다. UTM 필드가 자동으로 나타납니다.

영업

Salesforce 데이터 연결을 사용하여매핑 유형 NS 고급의그런 다음 UTM 필드를 Salesforce의 적절한 데이터 개체에 매핑할 수 있습니다.

Eloqua

Eloqua 데이터 연결을 사용하면 추가 설정이 필요하지 않습니다. UTM 필드가 자동으로 나타납니다.

Marketo

Marketo에서 생성 한 사용자 지정 활동에 추가 할 수있는 필드가 충분하지 않기 때문에 UTM 필드는 Marketo 연결에서 지원되지 않습니다.

캠페인 모듈을 사용하여 데이터 내보내기

캠페인 데이터를 내보내고 선택할 때 UTM 필드가 포함됩니다. 이벤트 보기 . 데이터 내보내기에 대한 정보는 다음을 참조하십시오. 캠페인 모듈에서 캠페인 데이터 내보내기 .

Audience API를 사용하여 UTM 필드에 액세스

UTM 필드는 Audience API를 사용하여 액세스 할 수 있습니다. 자세한 내용은 Audience API 설명서를 참조하십시오 .
