사용자 관리 페이지는 Studio 사용 권한이있는 사용자를 관리합니다. 사용자 관리 페이지에 액세스하려면 관리자 역할이있는 사용자 여야합니다.

사용자 관리 페이지에 액세스하려면 다음과 같이 하십시오.

  1. Studio에 로그인합니다.
  2. 탐색 헤더에서 ADMIN을클릭한 다음사용자 관리를 클릭합니다 .

사용자 관리 페이지가 표시됩니다.

  • 성 이름
  • 이메일 주소
  • 역할-표준 또는 관리자
  • 모듈 액세스 - 사용자가 액세스할 수 있는 각 모듈은 녹색 확인 표시()
  • 마지막 로그인-사용자가 Video Cloud Studio에 마지막으로 로그인 한 날짜 및 시간입니다. 현재 계정뿐만 아니라 모든 Video Cloud Studio 계정에 대한 마지막 로그인입니다.

사용자 관리 페이지를 사용하여 다음을 수행 할 수 있습니다.

  • 새 사용자 만들기
  • 기존 사용자 편집
  • 사용자 삭제
  • 기본 사용자 변경

몇 명의 사용자를 가질 수 있습니까?

생성 할 수있는 사용자 수는 보유하고있는 계정의 종류에 따라 다릅니다. 사용자를 더 추가해야하는 경우 더 많은 사용자를 제공하는 플랜으로 업그레이드 할 수 있습니다. 한도는 항상 존재하는 총 사용자 수에만 적용됩니다. 10 명의 사용자를 생성하고 5 명을 삭제하고 3 명의 사용자를 더 생성하면 8 명의 사용자가있는 것으로 간주됩니다. 사용자 페이지에는 자격이 부여 된 사용자 수와 생성 할 수있는 추가 사용자 수가 표시됩니다.

추가 사용자 구매

사용자를 더 추가해야하는 경우 더 많은 사용자를 제공하는 플랜으로 업그레이드 할 수 있습니다. Pro 및 Enterprise 수준의 계정은 사용자 추가에 대한 정보를 계정 관리자에게 문의해야합니다.

새로운 사용자 생성

새 사용자를 생성하려면 다음과 같이 하십시오.

  1. 들어가다이메일 , 이름그리고성사용자의.
  2. 사용자에게 역할을 할당합니다.
    • 관리자 - 사용자가 ADMIN메뉴에 액세스할 수 있습니다 (예: 사용자 계정 생성, 사용자 정의 수집 프로필, 사용자 지정 필드, 미디어 공유 구성 가능)
    • 표준 - 사용자가 ADMIN메뉴에 액세스할 수 없습니다.
  3. 사용자가 액세스 할 수있는 모듈을 선택하십시오. 모듈을 끄면 사용자가 액세스 할 수있는 UI 섹션 만 제한됩니다.
  4. 사용자추가를 클릭합니다 .

사용자는 계정에 추가되었음을 알리는 이메일 메시지를 받게됩니다.

기존 사용자 편집

기존 사용자를 편집하려면 다음과 같이 하십시오.

  1. 편집 할 사용자를 클릭하십시오. 사용자 세부 정보는 페이지 상단에 나타납니다.
  2. 사용자 세부 정보를 편집합니다.
    참고: 로그인한 사용자는 편집할 수 없습니다.
  3. 저장을클릭하여 변경 사항을 저장합니다.

사용자 삭제

사용자를 삭제하려면 다음과 같이 하십시오.

  1. 삭제 아이콘() 사용자와 연결됩니다.
  2. 확인을클릭하여 삭제를 확인합니다.

기본 사용자 변경

각 계정의 한 사용자가 기본 사용자로 지정됩니다. 주 사용자는 항상관리자역할을 가지며 브라이트코브 지원에서만 삭제할 수 있습니다.

특정 시점에 주 사용자를 변경해야 하는 경우 새 기본 사용자를 선택한 다음 이 사용자를 이 계정의 기본 관리자로만들기를 클릭합니다 .

기본 사용자 변경을 확인하라는 메시지가 표시됩니다.

새 기본 사용자에게표준역할이 있는 경우 기본 사용자가 될 때관리자역할이 할당됩니다.

여러 계정으로 작업

추가 계정을 추가 할 때 일반적인 오해는 첫 번째 Video Cloud 계정에서했던 것처럼 추가로 10 명의 사용자를 확보한다는 것입니다. 추가 Video Cloud 계정에는 추가 사용자 계정이 포함되지 않습니다. 예를 들어 10 명의 사용자가 첫 번째 계정에 추가되고 동일한 10 명의 사용자가 두 번째 계정에 추가 된 경우 총 사용자 수는 여전히 10 명입니다. 10 명의 고유 사용자는 라이선스 계약에 따라 지정되며 단일 고유 사용자 ID는 여러 계정에서 사용할 수 있으며 여전히 한 명의 사용자로 계산됩니다.

계정이 여러 개인 경우 각 계정에 개별적으로 사용자를 추가해야 합니다. 계정 A와 계정 B의 2개 계정이 있다고 가정합니다. 사용자가 계정 A에 추가되면 자동으로 계정 B에 액세스할 수 없습니다. 액세스해야 하는 각 계정에 사용자를 추가해야 합니다.

여러 계정으로 작업하는 방법에 대해 자세히 알아보려면여러 비디오 클라우드 계정으로 작업하기 .
