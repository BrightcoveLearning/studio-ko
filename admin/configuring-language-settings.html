참고: 이 기능은 모든 고객이 사용할 수 있지만 기본적으로 활성화되어 있지 않습니다. 계정에 대해 설정하려는 경우 계정 관리자에게 문의하십시오.

언어 설정은 기본 언어를 정의한 다음 추가 메타 데이터 언어를 추가하고 관리하는 곳입니다. 기본 언어는 계정에서 현재 동영상 메타 데이터에 사용하는 언어입니다. 계정에 다른 언어를 추가하기 전에 기존 메타 데이터가 기본 언어로 정의한 언어와 연결됩니다. 기본 언어는 미디어 모듈에서 비디오 속성을 편집 할 때 처음에 표시되는 언어입니다.

기본 언어 설정

기본 언어를 설정하려면

  1. 비디오 클라우드 스튜디오에 로그인합니다.
  2. 탐색 헤더에서 ADMIN 을클릭한 다음언어 설정 을 클릭합니다 . ADMIN 메뉴에 액세스하려면 계정 관리자 여야합니다.
  3. 기본 언어를 입력하십시오. 입력 할 때 일치하는 언어 목록이 나타납니다.
  4. 언어를 클릭하여 선택하십시오.
  5. 언어저장을 클릭합니다 .
기본 언어를 변경하려면 언어를 클릭하고 새 언어를 선택한 다음저장을 클릭합니다 .

추가 언어 추가

기본 언어를 저장 한 후 추가 메타 데이터 언어를 추가 할 수 있습니다. 추가 언어를 추가하면 미디어 모듈에 추가 다국어 메타 데이터를 입력 할 수 있습니다. 다른 언어를 추가하려면 다음 단계를 따르십시오.

  1. 언어추가를 클릭합니다 .
  2. 언어를 입력하십시오. 입력 할 때 일치하는 언어 목록이 나타납니다.
  3. 언어를 클릭하여 선택하십시오.
  4. 언어저장을 클릭합니다 . 언어가 추가 언어 테이블에 추가됩니다.
  5. 필요에 따라 언어를추가하려면 언어 추가를 클릭합니다.

언어 이름을 편집하려면 (미디어 모듈에 표시됨), 언어 이름을 클릭하고 변경한 다음저장을 클릭합니다 . 언어를 삭제하려면 삭제 아이콘().

참고: 사용중인 언어를 삭제하면 해당 언어와 관련된 모든 메타 데이터가 숨겨집니다. 연결된 메타데이터를 복원하려면 언어를 다시 추가합니다.
