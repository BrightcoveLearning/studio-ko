인제스트 프로필에는 비디오가업로드 모듈아니면 그동적 수집 API .

인제스트 프로파일 관리 페이지에 액세스하려면 ADMIN링크를 클릭한 다음 프로파일수집 을 클릭합니다 . Ingest Profiles 관리 페이지에 액세스하려면 관리자 역할이있는 Video Cloud Studio 사용자 여야합니다.

수집 프로필 목록이 나타납니다.

참고: 클릭하면 API 이름 표시헤더의 링크는 수집 프로필의 내부 이름을 표시합니다. 내부 이름은 API 호출에서 인제스트 프로필 이름을 사용할 때 사용됩니다.

페이지에는 다음 정보가 표시됩니다.

  • 프로필 이름 및 설명
  • 프로필 버전 -사용되는 트랜스 코딩 유형을 나타냅니다. 표시되는 유형은 Video Cloud 계정이 구성된 방식을 기반으로합니다.
    • 동적 전달 -Dynamic Delivery 트랜스 코딩이 사용됩니다. 동적 전달에 대한 자세한 내용은 동적 배달개요를 참조하십시오 .
    • 컨텍스트 인식  - 컨텍스트 인식 트랜스코딩이 사용됩니다. 컨텍스트 인식 인코딩에 대한 자세한 내용은 컨텍스트 인식 인코딩개요를 참조하십시오 .
  • 유형 -수집 프로필 유형
    • 표준 VOD -이것이 Video Cloud에서 제공하는 기본 프로필임을 나타냅니다. 이 유형의 프로필은 편집하거나 삭제할 수 없습니다.
    • 사용자 정의 VOD - 사용자 정의 인제스트 프로파일임을 나타냅니다 .
    • 스탠다드 라이브 -이것이 Video Cloud에서 제공하는 기본 라이브 프로필임을 나타냅니다. 이 유형의 프로필은 편집하거나 삭제할 수 없습니다.
    • 사용자 지정 라이브 - 사용자지정 라이브 인제스트 프로필임을 나타냅니다 .
  • 변환 -생성 될 수있는 비디오 변환 수입니다.
  • 표시 - 프로파일이표시되고 업로드 모듈에서 선택할 수 있는지 여부를 결정합니다.

Brightcove에서 제공한 수집 프로필에 대한 자세한 내용은유형 NS 표준 VOD또는스탠다드 라이브 ), 다음 문서 중 하나를 참조하십시오.

  • 동적 전달 또는 컨텍스트 인식 프로필 버전 - 동적 전달 및 컨텍스트 인식 인코딩을위한 표준 수집 프로필
  • 커스텀 라이브 유형 - 표준 Live Ingest 프로필

Ingest Profiles 관리 페이지를 사용하여 다음을 수행 할 수 있습니다.

  • 새 수집 프로필 만들기
  • 수집 프로필 편집
  • 수집 프로필 삭제
  • 기본 수집 프로필 변경
  • 수집 프로필 숨기기 / 표시

새 수집 프로필 만들기

참고: 수집 프로필을 만들고 편집하는 기능은 엔터프라이즈 및 프로페셔널 계정이 있는 게시자만 사용할 수 있습니다.

새 인제스트 프로파일을 만들려면 [ 새 프로필] 을 클릭합니다 .

새 수집 프로필을 만드는 프로세스는 만드는 프로필 유형에 따라 다릅니다. 사용자 지정 수집 프로필을 만드는 방법에 대한 자세한 내용은 다음 문서 중 하나를 참조하십시오.

  • Dynamic Delivery를위한 수집 프로필 생성
  • 컨텍스트 인식 인코딩을위한 수집 프로파일 생성
  • 사용자 정의 라이브 인제 스트 프로파일 생성

수집 프로파일 편집

수집 프로파일 관리 페이지는 수집 프로파일을 편집하는 데 사용할 수 있습니다. 기본 수집 프로파일 세트 ( 표준 VOD 표시 ) 는 편집할 수 없습니다.

인제스트 프로파일을 편집하려면 다음과 같이 하십시오.

  1. Studio 헤더에서 ADMIN링크를 클릭합니다.
  2. 인제스트 프로파일링크를 클릭합니다.
  3. 편집 할 수집 프로파일을 클릭하십시오.
  4. 프로필 편집기를 사용하여 프로필 정의를 편집합니다.
  5. 저장을 클릭합니다 .

수집 프로필 삭제

Ingest Profiles 관리 페이지를 사용하여 더 이상 필요하지 않은 수집 프로파일을 삭제할 수 있습니다. 기본 수집 프로파일 집합 ( 표준 VOD ) 은 삭제할 수없지만 필요하지 않은경우 숨길 수 있습니다.

인제스트 프로파일을 삭제하려면 다음과 같이 하십시오.

  1. Studio 헤더에서 ADMIN링크를 클릭합니다.
  2. 인제스트 프로파일링크를 클릭합니다. Ingest Profiles 관리 페이지가 열리고 계정에 대해 생성 된 수집 프로필이 표시됩니다.
  3. 삭제 아이콘() 수집 프로필 옆에 있는 프로필 이름을 클릭한 다음프로필 삭제 .
  4. 프로필 삭제를 클릭하여삭제를 확인합니다 .

기본 수집 프로필 선택

비디오가 업로드되는 시점및 Social 모듈을 사용하여클리핑할 때 기본 수집 프로필을 구성할 수있습니다. 업로드 기본 프로필은 다음으로 표시됩니다. 프로필 이름 옆에 소셜 클리핑 기본 프로필은 다음으로 표시됩니다. 프로필 이름 옆에

업로드/클리핑을 위한 기본 인제스트 프로필을 선택하려면 다음과 같이 하십시오.

  1. Studio 헤더에서 ADMIN링크를 클릭합니다.
  2. 인제스트 프로파일링크를 클릭합니다. Ingest Profiles 관리 페이지가 열리고 계정에 대해 생성 된 수집 프로필이 표시됩니다.
  3. 기본값으로 설정하려는 프로필의 수집 프로필 이름을 클릭합니다.
  4. 페이지 상단의업로드또는소셜 클리핑링크를 클릭하여 현재 프로필을 기본값으로 설정합니다.
참고: 기본 인제스트 프로파일을 선택할 때 프로필이표시되고 숨겨져 있지 않은지 확인해야합니다.
참고: Dynamic Delivery 프로필 만 소셜 클리핑 기본 프로필로 설정할 수 있습니다. CAE 프로파일은 지원되지 않습니다.

수집 프로필 숨기기 및 표시

인제스트 프로파일은 프로필과 연결된 슬라이더를숨기기위치로 클릭하여 숨길 수 있습니다. 필요하지 않을 수있는 기본 프로필을 숨기는 데 유용합니다.

숨겨진 인제 스트 프로필은 업로드 모듈의 인제 스트 프로필 목록에 표시되지 않습니다.
