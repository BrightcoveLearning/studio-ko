Video Cloud Studio를 사용하여 계정에 대한 청구 정보를 볼 수 있습니다. 결제 페이지에 액세스하려면 관리자 역할이있는 Video Cloud Studio 사용자 여야하며 계정과 연결된 연간 계약이 있어야합니다. 결제 페이지에 액세스하려면 다음과 같이 하십시오.

  1. 비디오 클라우드 스튜디오에 로그인합니다.
  2. 탐색 헤더에서 ADMIN 을클릭한 다음결제를 클릭합니다 .

기본 결제 페이지가 표시됩니다.

메모:
  • 페이지 상단에결제 계정드롭다운 목록이 나타나면 브라이트코브와 여러 계약이 있음을 나타냅니다. 드롭 다운을 사용하여 결제 세부 정보를 볼 계약을 선택합니다.
  • 하나 이상의 Zencoder 계정과 Video Cloud 계정이있는 경우 VOD Zencoder 사용 시간 항목이 Video Cloud 청구서에 나타날 수 있습니다. 이 정보는 귀하의 청구에 대한 완전한 그림을 제공하기 위해 편의를 위해 포함되어 있습니다. 이러한 요금은 Video Cloud와 관련이 없습니다.
  • 동영상을 삭제하면 해당 결정을 되돌려 야 할 경우를 대비하여 12 일 동안 시스템에 보관됩니다. 따라서 삭제 된 동영상은 삭제 된 후 12 일 동안 관리 콘텐츠에 계속 반영됩니다.

만료 된 계정

만료된 오래된 계정이 있는 경우결제 계정드롭다운에서 계정을 선택할 수도 있습니다.

만료 된 계정의 경우 만료 된 계약의 마지막 기간을 사용할 수없는 문제가 있습니다. 위의 예에서 만료된 계약을 선택하면 2020년 8월 데이터를 사용할 수 없습니다. 이 정보가 필요한 경우 계정 관리자 또는 Brightcove 지원에 문의하십시오.

결제 측정 항목

결제 페이지에 표시되는 데이터는 보유한 계정 수와 계정 권한에 따라 다릅니다. 예를 들어 DRM을 사용하지 않으면 해당 정보가 표시되지 않습니다.

참고: 지난 달의 결제 정보는 현재 월 7일까지 사용할 수 없습니다.

계약 개요 섹션에는 다음과 같은 청구 지표가 표시 될 수 있습니다.

대역폭

대역폭 지표는 VOD Edge 대역폭과라이브에지 대역폭인 2개의 하위 측정법으로 구성됩니다. 총 대역폭 / 약정 대역폭을 표시합니다. 총 대역폭에는 비디오 및 플레이어로드가 포함됩니다. 이 메트릭 이름의 이름은 대역폭 사용량을 추적하기 위해 청구서에 표시되는 이름과 일치합니다 (즉, 대역폭으로 표시되지 않을 수 있음). 대역폭 양이 할당량을 초과하면 그래픽이 빨간색으로 표시됩니다. 막대형 차트는계약 날짜또는월 누계 대역폭을 보는 데 사용할 수있습니다.

참고: 대역폭 데이터는 일반적으로 더 빨리 제공되지만 최대 3-5 일까지 지연됩니다. 데이터는 CDN의 정보에 따라 부분적으로 의존하기 때문에 통제 범위를 벗어나는 지연이 발생할 수 있습니다.

스트림

Streams 메트릭은 총 스트림 시작 / 계약 된 스트림 시작을 표시합니다. 이는 Analytics 모듈의 동영상 조회수와 일치해야합니다. 스트림 시작 횟수가 할당량을 초과하면 그래픽이 빨간색으로 표시됩니다. 막대형 차트는 스트림을 보기 위해 (수천 단위) 계약 날짜또는월 누일을 볼 수있습니다.

참고: 스트림 시작 횟수는 Analytics 모듈의 비디오 보기 수에 해당합니다.
참고: 결제 데이터와 분석 데이터를 비교할 때 불일치가있을 수 있습니다. 이는 조정 된 청구 데이터를 조정되지 않은 분석 데이터와 비교하기 때문일 수 있습니다. 조정 프로세스에는 CDN에서 제공하는 서버 로그가 필요하며 서버 로그를 가져오고 데이터를 처리하는 데 지연될 수 있습니다.

암호화 라이선스

암호화 라이선스 메트릭은 총 DRM 라이선스 / 계약 라이선스를 표시합니다. 사용 된 라이센스 수가 할당량을 초과하면 그래픽이 빨간색으로 표시됩니다. 막대형 차트는계약 날짜또는월 누계 사용권 수를 보는 데 사용할 수있습니다.

플레이어 사전 로드 설정에 관계없이 비디오 로드 지연을 방지하기 위해 비디오가 스트리밍되기 전에도 암호화가 항상 로드됩니다. 이러한 이유로 결제 지표에 Streams보다 많은 암호화 라이선스가 표시될 수 있습니다.

저장

스토리지 메트릭은 총 스토리지 / 계약 스토리지를 표시합니다. 사용 된 저장 용량이 할당량을 초과하면 그래픽이 빨간색으로 표시됩니다. 막대 차트는계약 날짜또는월 누계 사용된 저장소를 보는 데 사용할 수있습니다.

메모:

  • 저장소는 Professional 및 Enterprise 계정에만 표시됩니다.
  • 비디오 렌 디션 / 자산 및 디지털 마스터는 별도로 저장되지만 스토리지 메트릭은 렌 디션 / 자산 및 디지털 마스터 모두에 대한 총 결합 스토리지를 나타냅니다.

트랜스 코딩

트랜스 코딩은 비디오 파일을 추가 비디오 형식 또는 비트 전송률로 변환하는 것과 관련하여 Brightcove에서 처리 한 총 출력 GB 양을 나타냅니다. 사용 된 트랜스 코딩이 할당량을 초과하면 그래픽이 빨간색으로 표시됩니다. 막대 차트는계약 날짜또는월 누계를 사용한 트랜스코딩을 보는 데 사용할 수있습니다.

오리진 제공 대역폭

전달 된 바이트에 대한 대역폭 사용량 (BYO CDN 게시자에게 적용됨).

게재 된 광고

게재 된 총 광고 수입니다. 이것은 또한 제품의 전반적인 가치와 관련이 있습니다. 이 측정 항목은 또한 라이브 모듈을 사용하는 게시자를위한 ROI 관련 정보 또는 다운 스트림 자료에 대한 데이터를 제공합니다.

살아있는 대역폭

콘텐츠를 트랜스 코딩 한 입력 시간.

Brightcove 라이브 출력 계층

Brightcove Live Output Tiers 메트릭은 총 라이브 시간 / 약정 된 라이브 시간을 표시합니다. 시간이 할당량을 초과하면 그래픽이 빨간색으로 표시됩니다. 막대형 차트는계약 누계또는월 누계 시간을 보는 데 사용할 수있습니다. 이 측정 항목을 선택하면 각 계층의 출력 시간이 막대 차트 위에 표시됩니다.

사용 된 라이브 모듈 시간과 라이브 대역폭이 결제 UI에 표시 되려면 이벤트 후 3 일이 걸립니다. 수집 계층 별 청구는 다음과 같습니다.

  • Tier 1은 1.0x 시간에 청구됩니다.
  • Tier 2는 1.5x 시간에 청구됩니다.
  • Tier 3는 2.0x 시간에 청구됩니다.

사용자 지정 라이브 인제스트 프로파일을사용하는 경우 계층은 라이브 프로필의 총 비트율에 의해 결정됩니다. RTMP 출력을 사용하면 인코더가 종료 된 경우 재 연결 제한 시간을 포함하여 작업이 시작될 때부터 종료 될 때까지 청구가 시작됩니다.

브라이트코브 라이브 DVR 시간

NS 브라이트코브 라이브 DVR 시간트랜스코딩된 입력 초 + DVR이 활성화된 라이브 비디오를 표시합니다. 시간이 할당량을 초과하면 그래픽이 빨간색으로 표시됩니다.

라이브 오리진

라이브 오리진 대역폭은 CDN의 캐시 미스로 정의되며 라이브 오리진에서 이행해야하는 요청입니다. 대역폭을 재판매 할 때 시스템에 비용 영향을 미치며 고객 당 마진과 플랫폼의 전반적인 수익성을 평가하여 향후 가격 논의를 주도하는 데 도움이됩니다.

참고: Live Origin 데이터는 결제 모듈에서 사용할 수 없습니다.

VOD 오리진

라이브 VOD 원본 대역폭 (BYO CDN).

계정 세부 정보보기

페이지의 계정 세부 정보 섹션에 선택한 메트릭에 대한 세부 정보가 표시됩니다. 보려는 메트릭을 선택하려면 계약 개요 섹션에서 항목을 클릭하거나 계정 세부 사항 섹션의 드롭 다운 목록을 사용하여 항목을 선택하십시오.

막대 차트가 업데이트되어 선택한 측정 항목에 대한 세부 정보가 표시됩니다. 이번 달과 지난 12 개월의 데이터가 표시됩니다. 차트 위로 마우스를 가져 가면 해당 월에 대한 세부 정보가 표시됩니다.

그러나 현재 달 위로 마우스를 가져가면 이는 ~ 아니다실시간 데이터. 현재 월의 데이터는 현재 날짜로부터 4일 전까지의 최근 데이터이며 추정치에 불과합니다.

모든 계정 목록이 막대 차트 아래에 표시됩니다. 기본적으로 여러 계정이있는 경우 모든 계정의 데이터가 표시됩니다. 단일 계정의 데이터를 보려면 계정의 라디오 버튼을 클릭하거나 행을 클릭하여 계정을 선택하십시오.

삭제 된 비디오 및 디지털 마스터

비디오, 디지털 마스터 및 기타 자산을 삭제할 때 실제 파일은 즉시 제거되지 않습니다. 두 가지 이유로 최소 7일 (때로는 최대 14일) 동안 보관합니다.

  • 실수로 삭제하여 복원하려는 경우
  • 비디오에 대한 메타 데이터 및 분석을 계속 얻을 수 있도록

실제 파일이 남아 있는 한, 삭제된 미디어 자산이 결제에 반영된것을볼 수 있습니다.

잠긴 계정

잠긴 계정의 저장 용량은 최대 60 일 동안 결제 데이터에 계속 반영됩니다.
