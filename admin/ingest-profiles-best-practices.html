몇 개의 프로필이 있어야합니까?

최대 200 개의 사용자 지정 프로필을 정의 할 수 있습니다. 그러나 수집 작업을위한 프로필을 선택할 때 혼동을 피하기 위해 일반적으로 필요한만큼만 만들어야합니다.

내 프로필을 어떻게 관리합니까?

당신은 사용할 수 있습니다 Studio의 인제스트 프로필 관리 페이지수집 프로필을 관리합니다. 이 인터페이스에서는 다음과 같은 다양한 옵션을 사용할 수 있습니다.

  • 기본 프로필 설정
  • 새 프로필 만들기
  • 기존 프로필에서 복제
  • 사용자 정의 프로파일 편집
  • 불필요한 프로필 제거

수집 프로필에 프로그래밍 방식으로 액세스해야 하는 경우 다음을 사용하여 사용자 지정 프로필을 생성, 읽기, 업데이트 및 삭제할 수 있습니다. 수집 프로필 REST API . 이렇게 하려면 OAuth 자격 증명을 설정해야합니다.

내 프로필을 어떻게 업데이트합니까?

위섹션을 참조하십시오 .

프로필에는 몇 개의 변환이 포함될 수 있습니까?

동적 배달프로필에는 최대 37개의 변환이 있을 수 있습니다. 사용자 정의 동적 배달 프로필의 경우 현재 사용 가능한미리정의된 변환만 선택할 수 있습니다.

사용자 정의 컨텍스트 인식 인코딩 프로필을 만드는 경우, min_renditions최소값이 1이고max_renditions최대값은 15입니다.

불필요한 변환은 관리 스토리지 비용에 추가되므로 게시하는 비디오 종류와 대상으로하는 대상 및 장치에 따라 재생 될 가능성이없는 변환을 만들지 마십시오.

계정의 표준 프로필은 일반적인 시청자 대역폭과 가장 일반적으로 사용되는 기기에서 우수한 시청 환경을 제공하도록 설계되었습니다.

사용자 정의 프로파일 이름 지정

사용자 지정 프로필 이름에는 공백이나 특수 문자가 포함되지 않아야합니다. 문자와 숫자를 사용하고 공백을 대시 (-) 또는 밑줄 (_)로 바꿉니다.

사용자 오류를 방지하려면 동영상을 업로드하는 사람들에게 프로필의 용도를 명확하게 전달하는 방식으로 프로필의 이름을 지정하십시오.

어떤 프로필을 기본값으로 설정해야합니까?

모든 표준 또는 사용자 지정 프로필은 Studio 또는 Ingest Profile API를 통해 기본값으로 설정할 수 있습니다. 기본값은 사용자가 특정 프로필을 지정하지 않은 경우에만 사용됩니다 (업로드 모듈에서 또는 Dynamic Ingest API 사용).

사용자가 특정 프로필을 지정하는 것을 잊은 경우에는 그다지 나쁜 선택이 아닌 것으로 기본 프로필을 설정하는 것이 좋습니다.
