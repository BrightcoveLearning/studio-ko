비디오 클라우드 스튜디오에 액세스하려면 https://studio.brightcove.com 로 이동합니다 . 이메일 주소와 비밀번호로 로그인하세요. 홈 페이지는 다음 기능을 제공합니다.

  • 시스템 알림보기
  • Video Cloud 설정 관리
  • 계정을 선택하고 사용자 프로필보기
  • Video Cloud 모듈에 액세스
  • Video Cloud 대시 보드보기

언제든지 최대 5 개의 세션을 가질 수 있습니다. 동일한 이메일 주소로 로그인 할 때마다 세션이 시작됩니다. 6 번째 세션이 시작되면 가장 오래된 세션이 무효화됩니다. Video Cloud Studio 세션 시간 제한은 24 시간입니다.

시스템 알림보기

Video Cloud Studio 헤더에 벨 아이콘이 표시됩니다. 사용 가능한 시스템 알림이 있을 때. 아이콘을 클릭하면 시스템 알림 목록이 표시됩니다.

Video Cloud Studio 헤더에 경고 아이콘이 표시됩니다. 시스템 문제가 있을 때. 아이콘을 클릭하면 Brightcove 시스템 상태페이지.

설정 관리

비디오 클라우드 스튜디오 설정 메뉴는 ADMIN메뉴에서 액세스할 수 있습니다. Video Cloud ADMIN 메뉴는 다음에 대한 액세스를 제공합니다.

  • 계정 정보 -현재 Video Cloud 계정에 대한 정보보기
  • 사용자 관리 -Video Cloud Studio 사용 권한이있는 사용자 관리
  • 계정 설정 -일반 계정 설정 구성
  • 업로드 설정 -업로드 모듈을 사용자 정의 할 수있는 옵션 제공
  • 수집 프로필 -Video Cloud 계정에서 수집 프로필 관리
  • 비디오 필드 -표준 비디오 필드를 필수로 표시하고 계정에 대한 사용자 지정 메타 데이터 필드를보고, 만들고, 수정합니다.
  • 언어 설정 -추가 메타 데이터 언어 지원 구성
  • 재생 권한 - 재생 권한 구성
  • 미디어 공유 -다른 계정에 동영상 공유
  • 서버 측 광고 설정 - 서버 측 광고 설정 구성
  • 플레이어 설정 -Player Management 및 Delivery System API에 대한 접근을위한 기본 사용자 이름 및 비밀번호 인증 사용 옵션 제공
  • 통합 -Zoom 회의를 Brightcove Live에 통합
  • 라이브 설정 -라이브 모듈에서 사용하는 설정 관리
  • API 인증 -API 자격 증명 생성 및 관리
  • 청구 -계정에 대한 결제 정보보기
참고: ADMIN 메뉴에 액세스하려면 계정 관리자여야 합니다.

계정 선택

여러 Video Cloud 계정에 액세스 할 수있는 경우 Video Cloud 계정 선택기를 사용하여 액세스하려는 Video Cloud 계정을 선택할 수 있습니다. 계정을 전환하려면이름 - 계정 Video Cloud Studio의 오른쪽 상단 모서리에 있습니다. 현재 계정은 계정 이름 앞에 점이 표시됩니다.

사용자 정보 및 암호를 업데이트하려면내 프로필링크를 클릭합니다. 자세한 내용은 프로필 정보관리를 참조하십시오 .

모듈 액세스

Video Cloud Studio 메뉴는 Studio 모듈에 대한 액세스를 제공합니다. 홈을 클릭하여 메뉴에액세스합니다.

표시되는 모듈은 Video Cloud 구독 및 사용자 프로필을 기반으로합니다.

  • 업로드 -Video Cloud 계정에 동영상 업로드
  • 미디어 -비디오 속성 설정, 폴더를 사용하여 비디오 구성 및 비디오 게시
  • 해석학 -비디오 및 플레이어 성능에 대한 분석보기
  • 플레이어 -플레이어 생성 및 스타일 지정 및 플레이어 용 게시 코드 생성
  • 라이브 -라이브 이벤트 생성 및 스트리밍
  • 클라우드 플레이 아웃 -비디오를위한 라이브 선형 채널 생성
  • 갤러리 -비디오 콘텐츠를위한 비디오 포털 생성
  • 사회적인 -소셜 미디어 계정에 비디오 배포
  • 운동 - Video Cloud의 비디오 분석을 마케팅 자동화 플랫폼에 연결

대시 보드보기

Video Cloud Studio 홈 페이지에는 Video Cloud 대시 보드가 표시됩니다. 대시 보드에는 계정의 동영상에 대한 높은 수준의 통계를 제공하는 일련의 "카드"가 포함되어 있습니다. 다음 카드를 사용할 수 있습니다.

  • 비디오 조회수 -선택한 기간 동안의 일일 동영상 조회수를 표시합니다. 클릭하면비디오 보기 세부 정보 보기링크를 클릭하면 동영상 실적을 볼 수 있는 실적 보고서가 열립니다.
  • 계정 요약 -계정의 총 동영상 및 플레이어 수와 최근 생성 및 업데이트 된 동영상 수를 표시합니다. 번호를 클릭하면 미디어 또는 플레이어 모듈이 열립니다.
  • 기기 종류 -선택한 기간 동안 동영상을 시청하는 데 사용되는 장치 유형을 표시합니다. 클릭하면기기 유형별 동영상 조회수 보기링크를 클릭하면 장치 유형별 비디오 성능을 볼 수 있는 성능 보고서가 열립니다.
  • 무엇이 새로운가요? -Video Cloud 제품 업데이트를 표시합니다. 전체 업데이트 목록은 Video Cloud Studio 릴리스 노트를 참조하십시오 .
  • 인기 급상승 동영상 -선택한 기간 동안 조회수가 가장 많은 동영상을 표시합니다. 카드 제목 링크를 클릭하면 동영상 실적을 볼 수있는 실적 보고서가 열립니다.
  • 지역보기 -선택한 기간 동안 시청자 위치를 표시합니다. 클릭국가별 동영상 조회수 보기국가별 동영상 조회수를 볼 수 있는 실적 보고서가 열립니다.
