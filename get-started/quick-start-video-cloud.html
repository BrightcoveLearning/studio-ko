이 Quick Start의 모든 항목을 검토 한 후 다음을 수행 할 수 있습니다.

  • Video Cloud Studio 내부 탐색
  • Video Cloud에 비디오 업로드
  • 비디오 속성 편집
  • 단일 비디오 플레이어 만들기 및 스타일 지정
  • 웹에 비디오 게시
  • 재생 목록 만들기
  • 재생 목록 플레이어 만들기
  • 웹에 재생 목록 게시

청중

평가 중이거나 Video Cloud를 처음 사용하는 사람.

전제 조건

  • Video Cloud 계정-평가판 계정을 등록 할 수 있습니다. 여기.
  • 사용할 수 있는 샘플 비디오가 있거나 샘플 비디오를다운로드할수 있습니다.

개요

이 개요의 각 항목은 Video Cloud를 사용하는 비디오 게시 워크 플로의 특정 부분을 다룹니다. 각 주제에는 개요 비디오와 따라 할 수있는 실습 단계가 있습니다. 모든 교육 비디오를 시청하고 실습 연습을 완료하지 않으려면 Video Cloud 기본주문형 교육 과정을 확인하십시오.

  • 비디오 업로드
  • 기본 비디오 속성 편집
  • 플레이어 생성
  • 선수 스타일링
  • 웹에 단일 비디오 게시
  • 수동 재생 목록 만들기
  • 스마트 재생 목록 만들기
  • 재생 목록 플레이어 만들기
  • 재생 목록 플레이어 게시

Video Cloud Analytics 모듈은이 Quick Start의 일부로 다루지 않습니다. Video Cloud 분석에 대해 알아보려면 다음을 시청하십시오. 비디오 클라우드 분석 소개주문형 교육 과정.

  • 사용 가능한 비디오 분석 유형에 대해 알아보려면비디오 클라우드 분석 소개주문형 교육 과정.
