단계

  1. 비디오 클라우드 스튜디오에서애널리틱스모듈을 엽니다.
  2. 딸깍 하는 소리약혼왼쪽 탐색에서
  3. 보고서의 기간을 지정하려면 페이지 상단의 날짜 선택기를 사용하십시오.
  4. 보기를 선택하십시오약혼 ~에 의해동영상치수.
  5. 1% 의뷰열을 클릭하여 데이터를 정렬합니다.
  6. 해당 비디오의 참여 데이터를 보려면 비디오 제목을 클릭하십시오. 그래프는 시간 경과에 따른 동영상 조회수를 보여줍니다. 그래프의 급증은 동영상의 일부가 계속해서 시청되었음을 나타냅니다. 갑작스러운 참여 감소도 조사해야합니다.
  7. 미리보기 플레이어에서 비디오의 해당 부분을 보려면 참여 선을 클릭하십시오.

페이지 상단의 검색 필드를 사용하여 특정 비디오를 검색 할 수 있습니다.
