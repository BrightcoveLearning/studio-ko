전제 조건

  • 당신은업로드된 동영상귀하의 Video Cloud 계정에

단계

  1. 비디오 클라우드 스튜디오에서미디어를클릭하여미디어모듈을 엽니다.
  2. 비디오 제목을 클릭하여 비디오 속성 페이지를 엽니 다.
  3. 비디오 속성은 섹션으로 구성됩니다. 비디오 속성을 편집하려면편집을클릭하거나 섹션을 두 번 클릭합니다.
  4. 저장을클릭하여 변경 사항을 저장합니다.
