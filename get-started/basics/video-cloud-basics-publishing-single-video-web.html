전제 조건

  • 당신은업로드된 동영상귀하의 Video Cloud 계정에
  • 플레이어를생성했습니다.

단계

  1. 미디어모듈을 엽니다.
  2. 동영상 섬네일 옆에있는 확인란을 선택하여 게시 할 동영상을 선택합니다.
  3. 게시 및 포함 > 웹으로 게시를 클릭합니다.
  4. 플레이어선택드롭다운 목록에서 사용할 플레이어를 선택합니다.
    참고: 브라이트코브 기본 플레이어는 기본적으로 모든 비디오 클라우드 계정에생성됩니다.
  5. 클릭 URL미리보기 플레이어 아래에서 새 브라우저 탭에서 엽니다. 비디오가 나타나고 재생할 수 있는지 확인하십시오.
  6. 소스 코드를 복사하여 웹 페이지에 붙여 넣습니다.
