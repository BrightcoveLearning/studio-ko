전제 조건

  • 당신은업로드된 동영상귀하의 Video Cloud 계정에
  • 수동또는스마트재생 목록을 만들었습니다.

단계

  1. 미디어모듈을 엽니다.
  2. 게시할 재생 목록을 선택합니다.
    • 측면 탐색에 재생 목록이 나타나면 재생 목록을 클릭한 다음 재생 목록게시를 클릭합니다.
    • 왼쪽 탐색 메뉴에서모든 재생 목록을클릭하고 재생 목록을 선택한 다음게시 및 포함 > 웹에 게시를 클릭합니다.
  3. 플레이어선택드롭다운 목록에서 사용할 플레이어를 선택합니다.
  4. 클릭 URL미리보기 플레이어 아래에서 새 브라우저 탭에서 엽니다. 재생 목록이 표시되고 비디오를 재생할 수 있는지 확인합니다.
  5. 소스 코드를 복사하여 웹 페이지에 붙여 넣습니다.
