단계

  1. 비디오 클라우드 스튜디오에 로그인합니다.
  2. 탐색 헤더에서 ADMIN을클릭한 다음사용자 관리를 클릭합니다 . 관리자 메뉴는 관리자 역할을 가진 사용자 만 사용할 수 있습니다.
  3. 들어가다이메일 , 이름그리고성사용자의.
  4. 사용자에게 역할을 할당합니다.
    • 관리자 - 사용자가 ADMIN메뉴에 액세스할 수 있음
    • 표준 - 사용자가 ADMIN메뉴에 액세스할 수 없습니다.
  5. 사용자가 액세스 할 수있는 모듈을 선택하십시오. 모듈을 끄면 사용자가 액세스 할 수있는 Studio 섹션이 제한됩니다.
  6. 사용자추가를 클릭합니다 .

사용자는 계정에 추가되었음을 알리는 이메일 메시지를 받게됩니다. 페이지 본문에는 계정에 액세스 할 수있는 모든 사용자가 나열됩니다.
